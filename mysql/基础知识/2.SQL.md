# 2.1SQl通用语法

- sql语句以分号结尾，可单行或多行
- sql语句可以使用空格增强可读性g
- 不区分大小写，但建议用大写
- 注释
	- 单行 --或#
	- 多行/* 注释内容*/
## sql分类
- 定义 DDL : 定义数据库对象
- 操作 DML ：数据的增删改查
- 查询 DQL ：查询数据库中表的记录
- 控制 DCL ： 创建数据库用户，控制访问权限


## DDL-数据库操作

- 查询所有数据库
>SHOW DATABASES；

- 查询当前正在使用的数据库
>SELECT DATABASE();

- 创建
>CREATE DATABASE [IF NOT EXISTS] 数据库名 [DEFAULT CHARSET 字符集][COLLATE 排序规则]；

- 使用
USE 数据库名；

## DDL-表操作
### DDL表查询操作

- 查询当前数据库的所有表 
>SHOW TABLES;

- 查询表结构
>DESC 表名；

- 查询指定的建表语句
>SHOW CREATE TABLE 表名；

### DDL表创建操作

- 创建
>CREATE TABLE 表名(
>	字段1 字段1类型[COMMENT 字段1注释]
>	字段2 字段2类型[COMMENT 字段2注释]
>	...
>	
>)[COMMENT 表注释]；

- 数据类型(tiny small mediu long)
	- 数值类型
		- INT
		- DOUBLE
	- 字符串
		- CHAR VARCHAR 定长就用CHAR 性能好
		- BLOB(存储二进制数据的)
		- TXT(长文本)
	- 日期类型
		- DATE(birthday 2025-10-01)
		- TIME
		- YEAR
		- TIMESTAMP



### DDL表修改操作

- 添加字段
>ALTER TABLE 表名 ADD 字段名 类型（长度）[COMMENT 注释][约束]；
- 修改数据类型
>ALTER TABLE 表名 MODIFY 字段名 新数据类型（长度）；
- 修改字段名和字段类型
>ALTER TABLE 表名 CHANGE 旧字段名 新字段名 类型（长度）[COMMENT 注释][约束]；
- 删除字段
>ALTER TABLE 表名 DROP 字段名；
- 修改表名
>ALTER TABLE 表名 RENAME TO 新表名；

### DDL表删除

- 删除表
>DROP TABLE [IF EXISTS] 表名；

- 删除指定表，并重新创建该表(只有表结构没有数据)
>TRUNCATE TABLE 表名；
