
# for 循环
## 三要素
```c++
for (初始化; 条件; 迭代) {
    // 循环体
}
```

```c++
for (int i = 0; i < 5; i++)
{
    Log("hello ,world")//Log is a printing function.
}

```
> i++ 和++i 区别

### 🔄 核心区别

|操作符|返回值|执行顺序|等效代码|
|---|---|---|---|
|`i++`|**递增前的值**|先返回值，后递增|`{ auto tmp = i; i += 1; return tmp; }`|
|`++i`|**递增后的值**|先递增，后返回值|`i += 1; return i;`|
```c++
int i = 5;
int a = i++;  // a = 5, 之后 i = 6
int b = ++i;  // i 先变为7, 然后 b = 7
```

# while 循环
## 组成部分
```c++
while (条件表达式) {
    // 循环体
}
```

```c++
int i = 0;
while (i<5)
{
	Log("hello world")；//Log is a printing fumction.
	i++；
}

```
---


# do while 循环
## 组成部分
```c++
do {
    // 循环体
} while (条件表达式);  // 注意结尾必须有分号
```

```c++
int i = 0;
do {
    Log("hello world");  // 先执行一次
    i++;                 
} while (i < 5);         // 后检查条件
```
---


# 三大循环对比

| 循环类型       | 适用场景          | 特点       |
| ---------- | ------------- | -------- |
| `for`      | 已知迭代次数/明确控制变量 | 紧凑的三要素控制 |
| `while`    | 条件主导的不确定次数循环  | 只有条件检查   |
| `do-while` | 至少执行一次的循环     | 后置条件检查   |
