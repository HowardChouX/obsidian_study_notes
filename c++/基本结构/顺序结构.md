**一、代码层面的顺序结构**

```cpp
// function.cpp（必须在前）
void printMessage() {
    std==cout << "Hello from function" << std==endl;
}

// main.cpp（在后）
int main() {
    printMessage(); // 正确：函数已定义
    return 0;
}
```

▌ 重要规则：

1. 函数/变量必须先声明后使用（通过头文件解决）
2. 编译阶段顺序：
    
    ```bash
    g++ -c function.cpp  # 先编译函数实现
    g++ -c main.cpp      # 再编译主程序
    g++ *.o -o program   # 最后链接
    ```
