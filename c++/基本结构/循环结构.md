# for 

### 核心结构

```cpp
for (初始化; 循环条件; 迭代操作) {  
    // 循环体代码  
}  
```

**执行顺序**：

1. 执行**初始化**（仅一次）
2. 检查**循环条件** → 若为 `true` → 执行循环体 → 执行**迭代操作** → 重复步骤2
3. 若条件为 `false` → 退出循环

---

### 与 `while` 循环对比

|特性|`for`|`while`|
|---|---|---|
|**适用场景**|已知循环次数/明确迭代步骤|未知循环次数/条件驱动|
|**变量作用域**|初始化变量通常只在循环内可见（C++11起支持循环内声明）|条件变量需在外部声明|

---

### 经典应用场景

1. **遍历数组**

```cpp
int arr[] = {2,4,6,8};  
for (int i = 0; i < 4; i++) {  
    cout << arr[i] << " ";  // 输出：2 4 6 8  
}  
```

2. **生成数列**

```cpp
for (int n=1, sum=0; n<=10; sum+=n, n++) {  
    // 计算1到10的累加和  
}  
```

3. **嵌套循环（如乘法表）**

```cpp
for (int i=1; i<=9; i++) {  
    for (int j=1; j<=i; j++) {  
        cout << i << "*" << j << "=" << i*j << "\t";  
    }  
    cout << endl;  
}  
```
