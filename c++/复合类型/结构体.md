

### 结构体组成

```c++
struct inflatable//结构体外部声明
{
const char name[20]//默认为public，对name字符数组初始化；
std::string name;//使用string对象初始化；
};

int main()
{
inflatable gifts[100] = 
{

	{"apple"},
	{"bird"}
};//创建一个包含100个inflatable结构的数组

}
```

## 与class类对比
**1. 类型本质**

- `struct` 是**值类型**（栈内存分配）
- `class` 是**引用类型**（堆内存分配）

**2. 内存分配**

- `struct` 直接存储数据本身
- `class` 存储对象引用地址

**3. 继承性**

- `struct` 不能继承，只能实现接口
- `class` 支持继承和多态

**4. 默认值**

- `struct` 不能为 null（除非用 `Nullable<T>`）
- `class` 默认值为 null

**5. 初始化器**

- `struct` 必须显式初始化所有字段
- `class` 字段有默认初始值

**6. 复制行为**

- `struct` 赋值会产生**副本**
- `class` 赋值会复制**引用**

**7. 使用场景**

- 用 `struct` 当：  
    ✓ 小型数据结构（建议小于 16 字节）  
    ✓ 需要频繁创建/销毁  
    ✓ 需要值语义（如坐标、颜色）
    
- 用 `class` 当：  
    ✓ 需要继承或多态  
    ✓ 表示逻辑实体（如用户、订单）  
    ✓ 需要引用语义
