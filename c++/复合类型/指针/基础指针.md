# 定义
## 声明

> 指针是存储内存地址的变量

```c++
int* ptr;  // 声明一个整型指针
```
---

## 取址运算符&

```c++
int num = 10;
int* ptr = &num;  // ptr 存储 num 的内存地址
```

---
## 解引用运算符*

```c++
cout << *ptr;  // 输出 10（通过指针访问数据）
*ptr = 20;     // 修改 num 的值为 20
```

---


# 指针操作
## 指针算数

> 适用于连续内存，如数组。

```c++
int arr[3] = {1,2,3};
int* p = arr;     // p 指向 arr[0]
cout << *(p+1);   // 输出 arr[1] 的值 2

```
---

## 动态分配内存

```c++
int* p = new int;    // 申请堆内存
*p = 5;
delete p;           // 必须手动释放内存
```
---

## 指针与数组

```c++
char str[] = "Hello";
char* p = str;      // 指针指向数组首地址
while (*p != '\0') 
{
    cout << *p++;
}
```

---

# 特殊指针类型

## 空指针

```c++
int* p = nullptr;  // 明确表示指针未指向任何地址
```
---

## 指向指针的指针（双指针）

```c++
int a = 10;
int* p = &a;
int** pp = &p;     // 二级指针
```

---

## 函数指针

```c++
void func(int x) { /*...*/ }
void (*fp)(int) = func;  // 函数指针赋值
fp(42);                  // 通过指针调用函数
```
---

 
# 智能指针(避免内存泄漏)
- **`unique_ptr`**（独占所有权）
```c++
std==unique_ptr<int> uptr = std==make_unique<int>(10);
```

- **`shared_ptr`**（共享所有权，引用计数）

```c++
std==shared_ptr<int> sptr1 = std==make_shared<int>(20);
auto sptr2 = sptr1;  // 引用计数+1
```

- **`weak_ptr`**（解决 `shared_ptr` 循环引用问题）

```c++
std::weak_ptr<int> wptr = sptr1;
```
