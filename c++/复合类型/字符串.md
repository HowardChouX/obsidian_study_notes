## 字符串字面量
**字符串字面量特性**  
`"Hello"` 是 `const char[6]` 类型的字符串字面量：
    
	存储在只读内存段
    不可修改内容
    可赋值给 `const char*`

```c++
//字符串字面量
const char* cat = "XioaHei"//存储为字符数组，隐式自动添加空终止符"\0"
char* name = "xaiohei" // 错误：C++11 起禁止将字符串字面量赋给非 const 指针
```

| 特性     | `const char*` 指针 | `char[]` 数组 |
| ------ | ---------------- | ----------- |
| 内存区域   | 只读数据段            | 栈内存         |
| 可修改性   | 不可修改             | 可修改         |
| 生命周期   | 整个程序运行期          | 所在作用域内      |
| 典型使用场景 | 常量字符串            | 固定长度可修改字符串  |

# 字符串赋值，拼接和附加
```c++


//字符串数组赋值
char char1[6] = "hello"
//string类赋值
std::string str1[6] = "hello"
//数组不可以赋值给另一个数组，但string可以赋值给另一个string类
std::string str2[6] = str1
//附加
str1 += str2;

using namespace std::string_literals;
std::string name0 = "Luo"s + "XiaoHei"

```


```c++

//输出流拼接
std::cout << "line1\n"
	 << "line2\n"
	 << "line3\n"
std::cout << "line1" "line2.\n"
std==cout << "line"s + "line2\n"//s 后缀是 C++11 引入的**用户定义字面量**，将字符串字面量转换为 `std==string` 对象。`+` 操作符：`std::string` 重载了 `+`，允许与 `const char*`（即 `"line2\n"`）拼接。

```

# 其他形式的字符串面值


| 字面量格式       | 类型               | 字符类型       | 所需命名空间                           | byte |

# 原始字符串（raw)
```c++

//用R换行省略"\n"

line2
line3
)";

//上面的R换行等价于
const char* example = "line1\n"
```
