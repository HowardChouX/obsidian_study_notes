
# 传统枚举

```c++
enum Color 
{
    Red,    // 0
    Green,  // 1
    Blue    // 2
};

Color c = Red;
```
**特点**：

- 枚举值直接暴露在外部作用域
- 隐式转换为整型
- 无法指定底层数据类型（默认为 int）
- 不同枚举类型之间可以比较（存在类型安全问题）

---
# 强类型枚举

```c++
enum class LogLevel : uint8_t 
{ // 可指定底层类型
    Debug = 0x1,
    Info  = 0x2,
    Error = 0x4
};

LogLevel level = LogLevel::Debug;
```
