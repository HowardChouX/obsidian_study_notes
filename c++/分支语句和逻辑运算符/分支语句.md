
# if 单分支
## 组成部分
```c++
if (condition)
{		//{}可以省略
	 //do something.
}
```

```c++
int x = 5;
bool comparsionResult = x == 5;
if (comparsionResult)
    Log("hello world");
```

# if-else if- else 多分支

# 组成部分
```c++
if (condition)
{
	//do something.
}
else if (contion)
{
	//do something.
}
else
{
	//do something.
}
```

```c++
const char* ptr = nullptr;
if (ptr)
	Log(ptr);
else if (ptr == "hello")
	Log("ptr is hello")
else
	Log("ptr is null")
```
