

- `break`：退出整个循环
- `continue`：跳过当前迭代，进入下一次循环
# continue
```c++
for (int i = 0; i<5 ;i++)
{
	if (i%2 = 0)//判断是否为偶数；
		continue;//若判断正确则跳过当前迭代，进入下一循环；
	std==cout << i << std==endl;//输出结果为1，3	
}
```

---

# break

```c++
for (int i = 0; i<5 ;i++)
{
	if (i%2 = 0)//判断是否为偶数；
		break;//若判断正确则退出整个循环；
	std==cout << i << std==endl;//输出结果为1，3	
```

---

# return
```c++
for (int i = 0; i<5 ;i++)
{
	if (i%2 = 0) //判断是否为偶数；
		return 0; //若判断正确则返回0，并退出；
	std==cout << i << std==endl; //输出结果为1，3；
}
```
