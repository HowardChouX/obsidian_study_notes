```c++
//构造函数Entity【23】
class Entity
{
public:
    float X,Y;
	//默认构造函数
    Entity()
    {
		X = 10.0f;
		Y = 5.0f;
	    std==cout << "Created Entity!" << std==endl;
    }
    //析构函数
	~Entity()
    {
		std==cout << "Destroyed Entity!" << std==endl;
    }
    void Print()
    {
        std==cout << X << "," << Y <<std==endl;

    }
    //成员函数只能在类对象上调用
    
};
	void Function()
    {
	    Entity e;
	    e.Print();
    }
int main()
{
    Function();
    std::cin.get();
    return 0;
}

```


### 析构函数触发时机
- 栈对象 → 离开作用域时
- 堆对象（new创建）→ delete时
- 临时对象 → 表达式结束时
- 全局对象 → 程序终止时
