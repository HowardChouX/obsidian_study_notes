### 一、基础输入输出

**头文件**：`<iostream>`

```cpp
cout << "输出";          // 标准输出
cin >> variable;        // 标准输入
cerr << "错误信息";      // 错误输出（无缓冲）
clog << "日志";         // 缓冲式日志输出
```

---

### 二、容器库

#### 1. **序列容器**

**头文件**：`<vector>`, `<array>`, `<list>`, `<deque>`

```cpp
vector<int> v{1,2,3};  
v.push_back(4);         // 尾部插入
v.pop_back();           // 尾部删除
v.size();               // 元素数量
v.empty();              // 判空
```

#### 2. **关联容器**

**头文件**：`<map>`, `<set>`

```cpp
map<string, int> m;
m.insert({"key", 42});  // 插入键值对
m["new_key"] = 100;     // 下标访问（自动创建）
m.count("key");         // 查找存在性
```

---

### 三、算法库

**头文件**：`<algorithm>`

```cpp
sort(vec.begin(), vec.end());                  // 快速排序
find(vec.begin(), vec.end(), target);          // 线性查找
reverse(str.begin(), str.end());               // 反转序列
max(a, b); min(a, b);                          // 最值
transform(src.begin(), src.end(), dest.begin(), // 元素转换
          [](int x){ return x*2; });
```

---

### 四、字符串处理

**头文件**：`<string>`

```cpp
string s = "Hello";
s += " World!";          // 拼接
s.substr(6, 5);          // 截取子串 → "World"
s.find("lo");            // 返回索引 3
stoi("42");              // 字符串转整数
to_string(3.14);         // 数值转字符串
```

---

### 五、数值处理

#### 1. **基础数学**

**头文件**：`<cmath>`

```cpp
sqrt(9.0);               // 3.0
pow(2, 3);               // 8.0
ceil(3.2); floor(3.7);   // 4.0 / 3.0
abs(-5); fabs(-3.14);    // 绝对值
```

#### 2. **数值算法**

**头文件**：`<numeric>`

```cpp
accumulate(vec.begin(), vec.end(), 0);       // 累加
partial_sum(vec.begin(), vec.end(), dest);   // 前缀和
gcd(a, b); lcm(a, b);       // C++17 起
```

### 六、智能指针

**头文件**：`<memory>`

```cpp
unique_ptr<int> p1(new int(42));  // 独占所有权
auto p2 = make_shared<string>("安全内存"); // 共享指针
weak_ptr<Obj> wp(p2);             // 观察指针（防循环引用）
```

### 七、实用工具

#### 1. **类型操作**

**头文件**：`<type_traits>`

```cpp
is_integral_v<int>;      // true
is_pointer_v<int*>;      // true
remove_reference_t<T>;   // 移除引用
```

#### 2. **时间处理**

**头文件**：`<chrono>`

```cpp
auto start = chrono::high_resolution_clock::now();
// ...代码...
auto duration = chrono::duration_cast<milliseconds>(end - start);
```

---

### 最佳实践建议

1. **优先使用 STL 容器**（如 `vector` 代替 C 风格数组）
2. **避免 `using namespace std`**（防止命名污染）
3. **组合使用算法与lambda**：
    
    ```cpp
    sort(vec.begin(), vec.end(), 
         [](auto a, auto b){ return a > b; }); // 降序排序
    ```
