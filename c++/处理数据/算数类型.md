####  `size_t` 的作用

- ​**类型定义**：  
    `size_t` 是 C/C++ 中的一种 ​**无符号整数类型**，通常用于表示：
    
    - 内存对象的大小（如数组长度、容器大小）
    - 数组或容器的索引（保证非负）
- ​**使用场景**：
    
    
    ```cpp
    for (size_t j = 0; j < primes.size(); ++j)
    ```
    
    - ​**`j` 的作用**：作为循环变量，遍历 `primes` 容器中已存储的质数。
    - ​**为什么用 `size_t`？**
        - `primes.size()` 返回的是容器的大小（`size_t` 类型），使用 `size_t` 避免类型不匹配的警告。
        - 无符号特性保证 `j` 始终非负，防止索引越界。
- ​**底层实现**：
    
    - 在 32 位系统上：`size_t` 等价于 `unsigned int`（4 字节，范围 0~4,294,967,295）。
    - 在 64 位系统上：`size_t` 等价于 `unsigned long`（8 字节，范围 0~18,446,744,073,709,551,61
