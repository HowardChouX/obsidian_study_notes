
1. **类/结构体**
    
    - 大驼峰式：`ClassName`
    - 示例：`class DataParser;` `struct HttpRequest;`
2. **函数/方法**
    
    - 小驼峰式：`functionName()`
    - 示例：`void calculateSum();` `string getUserId();`
3. **变量**
    
    - 小写 + 下划线：`variable_name`
    - 示例：`int total_count;` `double average_score;`
4. **常量**
    
    - 全大写 + 下划线：`CONSTANT_NAME`
    - 示例：`const int MAX_BUFFER_SIZE = 1024;`
5. **枚举值**
    
    - 全大写 + 下划线：`ENUM_VALUE`
    - 示例：
        
        ```cpp
        enum Color {
            RED = 0,
            BLUE_CHANNEL = 1
        };
        ```
        
6. **命名空间**
    
    - 全小写：`namespace_name`
    - 示例：`namespace file_system { ... }`
7. **宏定义**
    
    - 全大写 + 下划线（慎用）：`MACRO_NAME`
    - 示例：`#ifdef DEBUG_MODE`
8. **文件命名**
    
    - 小写 + 下划线：
        - 头文件：`data_parser.h`
        - 源文件：`network_utils.cpp`

**特殊约定**：

- 布尔变量加 `is/has/can` 前缀：`is_valid`, `has_permission`
- 私有成员变量加 `_` 后缀：`int counter_;`
- 避免缩写（除非广泛接受）：用 `index` 而非 `idx`
- 模板参数用 `T`/`U`：`template<typename T>`

**需要避免**：

```cpp
int I;          // 无意义单字母
double m_dData; // 匈牙利命名法（已过时）
char *strNAME;  // 混合大小写
```

**示例对比**：

```cpp
// 好的命名
class UserManager { 
public:
    void updateProfile(const string& new_name);
private:
    int login_count_;
};

// 差的命名
class usermgr {  // 缩写 + 全小写
public:
    void UpdPrf(string n);  // 过度缩写
private:
    int lc;       // 无意义缩写
};
```
