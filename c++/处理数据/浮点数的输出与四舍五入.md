# 输出流显示四舍五入

> cout << fixed << setprecision(2) << num << endl;

1. **`fixed`**
    
    - **作用**：强制浮点数以**定点表示法**（非科学计数法）输出。
    - **示例**：
        
        ```cpp
        double num = 12345.6789;
        cout << fixed << num;  // 输出 12345.678900（默认保留6位小数）
        ```
        
2. **`setprecision(2)`**
    
    - **作用**：与 `fixed` 配合时，指定小数点后保留**2位有效数字**。
    - **示例**：
        
        ```cpp
        double num = 3.1415;
        cout << fixed << setprecision(2) << num;  // 输出 3.14
        ```
        
3. **组合效果**
    
    - 浮点数始终显示为 `整数部分.小数部分` 形式，且小数部分严格保留两位（自动四舍五入）。
    - **示例**：
        
        ```cpp
        double a = 2.71828;    // 输出 2.72
        double b = 5;          // 输出 5.00
        double c = 1000.4999;  // 输出 1000.50
        ```


# 输出流截断不四舍五入

> cout << std::trunc (value\*100.0)/100.0 <<endl;

1. **缩放数值**：  
    `value * 100.0` 将小数点后两位移到整数部分  
    _示例_：3.1415 → 314.15
    
2. **截断处理**：  
    `std::trunc()` 直接舍弃小数部分  
    _示例_：314.15 → 314.0
    
3. **恢复量级**：  
    `/ 100.0` 将数值还原到原数量级  
    _示例_：314.0 → 3.14

#  `cout << fixed << setprecision(2)` 和 `round()` 


|特性|`fixed + setprecision(2)`|`round()`|
|---|---|---|
|**作用对象**|输出流的显示格式|数值本身的数学运算|
|**修改原值**|否（仅改变显示方式）|是（返回新值）|
|**精度控制**|控制小数点后位数|四舍五入到指定精度|
|**内存影响**|不改变变量存储值|生成新的数值|
|**后续运算影响**|无|影响后续所有计算|
|**头文件**|`<iomanip>`|`<cmath>`|
