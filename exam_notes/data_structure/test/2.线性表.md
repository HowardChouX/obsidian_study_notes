## 四、算法分析题（每题15分）

1. **顺序表删除所有值为x的元素**
   分析以下两种算法的区别：
   ```python
   # 算法1：前移法
   def Deletex1(L, x):
       k = 0
       for i in range(L.getsize()):
           if L[i] != x:
               L[k] = L[i]
               k += 1
       L.size = k
 
   # 算法2：计数法
   def Deletex2(L, x):
       k = 0
       for i in range(L.getsize()):
           if L[i] != x:
               L[i-k] = L[i]
           else:
               k += 1
       L.size -= k
   ```
   分析两种算法的时间复杂度和空间复杂度[1][25]。

2. **单链表查找中间位置元素**
   分析以下两种算法的区别：
   ```python
   # 算法1：计数法
   def Middle1(L):
       j = 1
       n = L.getsize()
       p = L.head.next
       while j <= (n-1)//2:
           j += 1
           p = p.next
       return p.data
 
   # 算法2：快慢指针法
   def Middle2(L):
       slow = L.head.next
       fast = L.head.next
       while fast != None and fast.next != None and fast.next.next != None:
           slow = slow.next
           fast = fast.next.next
       return slow.data
   ```
   分析两种算法的时间复杂度和空间复杂度[12][15]。

## 五、程序设计题（20分）

**有序单链表的二路归并**
有两个递增有序整数单链表A和B，设计一个算法采用二路归并方法将A和B的所有数据结点合并到递增有序单链表C中。要求算法的空间复杂度为O(1)[18]。

请写出算法的主要思路，并分析：
1. 算法的时间复杂度
2. 算法的空间复杂度
3. 如果要求不改变原链表A和B，算法应如何修改？

**提示**：参考有序单链表的二路归并算法[14][18]。

---

**参考答案要点：**

一、选择题：
1. D 2. C 3. A 4. B

二、填空题：
1. raᵢ，aᵢ₊₁> | aᵢ，aᵢ₊₁∈D，i=0，…，n-2}
2. i<j
3. 相反
4. next

三、简答题：
1. 空间：顺序表存储密度为1，链表小于1；顺序表需要预分配空间，链表动态分配。时间：顺序表随机存取O(1)，链表顺序访问O(n)；顺序表插入删除平均移动半个表，链表只需修改指针[7][19]。
2. 从头开始遍历原链表，将每个结点用头插法插入新链表。时间复杂度O(n)，空间复杂度O(1)[31]。
3. 区别：循环单链表尾结点指向头结点。尾结点判断条件：p.next==head[11][63]。
4. 使用双指针分别遍历两个多项式，比较指数大小，较大者加入结果；指数相同时系数相加，若不为0则加入结果[3][10]。

四、算法分析题：
1. 算法1：前移法，时间复杂度O(n)，空间复杂度O(1)。算法2：计数法，时间复杂度O(n)，空间复杂度O(1)。两种算法效率相同，但实现思路不同[1][25]。
2. 算法1：计数法，需要先遍历求长度，再遍历到中间位置，时间复杂度O(n)。算法2：快慢指针法，只需一次遍历，时间复杂度O(n)，但常数因子更小[12][15]。

五、程序设计题：
主要思路：使用三个指针p、q、t分别指向A、B、C的当前结点，比较p和q的值，将较小者链接到C的末尾。
1. 时间复杂度：O(n+m)，其中n、m分别为A、B的长度。
2. 空间复杂度：O(1)，只使用固定数量的指针变量。
3. 如果不改变原链表，需要创建新结点复制数据，此时空间复杂度为O(n+m)[14][18]。
