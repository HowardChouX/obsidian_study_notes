## 四、分析题（每题15分）

1. **时间复杂度分析**
   分析以下算法的时间复杂度：
   ```python
   def example_algorithm(n):
       sum = 0
       for i in range(n):
           for j in range(i, n):
               sum += 1
       return sum
   ```

2. **空间复杂度分析**
   分析以下算法的空间复杂度：
   ```python
   def matrix_addition(A, B, n):
       C = []
       for i in range(n):
           row = []
           for j in range(n):
               row.append(A[i][j] + B[i][j])
           C.append(row)
       return C
   ```

4. 从集合中删除元素

6. 求两个集合的交集
7. 求两个集合的差集

请写出该ADT的Python类定义，并简要说明每个方法的实现思路。
