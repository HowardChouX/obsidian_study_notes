### **第五章 生产与成本理论 核心知识点**

#### **第一节 企业、生产与成本**

1. **企业的本质**
   * **企业**：泛指能够做出统一生产和供给决策的基本单位[4]。是有投入才有产出，对要素的需求是生产的前提；销售商品或劳务，获取收益的同时承担风险[4]。
   * **企业的本质**：是可以降低交易费用，替代市场，实现资源有效利用的组织[9]。**交易费用**（又称交易成本）是指签订、监督、执行契约等所花费的成本[9]。企业将许多原本外在的市场交易转化为内部的交易关系或内部的合作关系，从而降低了交易费用[9]。

2. **企业的经济利润**
   * **利润** = 总收益 - 总成本[1]
     * **总收益**：企业出售其产品所得到的货币量[1]。
     * **总成本**：企业用于生产的投入品的市场价值[1]。
   * **会计利润**（正常利润）= 总收益 - 显性成本[1]
   * **经济利润**（超额利润）= 总收益 - 机会成本 = 总收益 - 显性成本 - 隐性成本[1]
   * **经济利润小于会计利润**[1]
   * **企业追求利润最大化**：利润 = 总收益 - 总成本；收益、成本与企业所处的市场结构密切相关[1]。企业更多的着眼于长期的利润。销售额、市场份额、销量等也会成为某个时期的目标，但从属于长期的利润目标[1]。

3. **企业面临的约束**
   * **技术约束**：生产要素的投入与产出量之间的物质技术关系[3]。
   * **经济约束**：各种生产要素的价格构成的成本问题[3]。

4. **机会成本角度的成本衡量**
   * **机会成本**是为了得到某种东西而为此放弃的东西[3]。机会成本即经济成本，包括显性成本和隐性成本[3]。
   * **显性成本**：即会计成本，需要企业支出货币的投入成本，包括雇员工资、购买原材料、燃料及添置或租用设备的费用、利息、保险费、广告费及租金等[3]。
   * **隐性成本**：投入生产的自有要素的货币价值[3]。
   * **私人成本与社会成本**：必须由经济个体承担的一切费用；社会因为个人的行为而承担的一切费用[7]。
   * **可回收成本与沉没成本**：已经发生在会计成本中的一般可以回收的成本；已经发生在会计成本中的不可能回收的成本[7]。

#### **第二节 短期生产**

1. **生产与生产函数**
   * **生产**：把**各种投入**转换为**产出**的一个过程[1]。生产需要时间，不是一蹴而就的；投入是产出的前提，产出是结果；生产有风险；如何转换体现了技术水平；投入具有多样性[1]。
   * **生产要素**：生产过程中的各种投入[1]。包括：
     * **资本**：生产过程中投入的物品和货币资金等[1]。
     * **劳动**：劳动者在生产过程中以体力或者脑力提供的各种服务。理论分析中是标准化的劳动[1]。
     * **企业家才能**：企业家发现市场机会，组织各种投入进行生产的能力[1]。
     * **土地**：泛指一切自然资源[1]。
   * **生产函数**：一定时期内，在**技术水平不变**的情况下，生产中所使用的各种生产要素的**数量**与所能生产的**最大产量**之间的关系[1]。因技术的改变而改变，更多强调理论分析上的意义[1]。

2. **短期与长期**
   * **短期**：一种或多种要素无法变更的时期[1]。
   * **长期**：所有投入品都可以变化的时期[1]。

3. **生产函数的具体形式**
   * **固定投入比例生产函数**：每单位产出需要的劳动、每单位产出需要的资本[10]。
   * **柯布-道格拉斯生产函数**：$Q = AL^αK^β$，其中α表示劳动的产出弹性系数，β表示资本的产出弹性系数，α和β的取值介于0-1之间[10]。

4. **短期生产函数**
   * 在一定的技术条件下，生产某产品的各投入要素中只有**一种**（通常是劳动）是**可变**的，生产中所使用的该要素的数量与所能生产的**最大产量**之间的关系[10]。
   * 表示为：$Q = f(L, \overline{K})$ 或 $Q = f(L)$[10]。

5. **总产量、平均产量与边际产量**
   * **总产量（TP）**：投入一定量的某种生产要素所生产出来的全部产量[10]。$TP_L = f(L)$[10]。
   * **平均产量（AP）**：平均每单位某种生产要素所生产出来的产量[10]。$AP_L = TP_L / L$[10]。
   * **边际产量（MP）**：在其他投入量不变情况下，增加一单位某种生产要素所增加的产量[10]。$MP_L = \Delta TP_L / \Delta L$[10]。

6. **三者的关系**
   * **总产量与平均产量的关系**：平均产量是从原点出发到总产量曲线上相应点的射线的斜率值，先递增后递减[10]。
   * **边际产量和总产量关系**：边际产量反应了总产量的变动率，是总产量曲线上对应点的切线斜率[10]。边际产量最大时，总产量上升最快；边际产量为0时，总产量最大[10]。
   * **边际产量和平均产量关系**：边际产量小于平均产量，拉动平均产量下降；边际产量大于平均产量，拉动平均产量上升；边际产量交于平均产量最大点[10]。

7. **边际报酬递减规律**
   * 在技术给定和生产的其他要素投入不变的情况下，连续增加某种可变投入使其边际产量增加到某一点，超过该点后，增加可变投入会使其边际产量减少[7]。
   * **主要原因**：可变要素与固定要素有一个**最佳配合比例**[7]。当可变要素较少时，固定要素相对较多，没有达到最佳比例，随着可变要素的增加，可使要素比例趋于合理从而使边际产量增加，提高总产量；当二者的组合达到最有效率的那一点以后，再增加可变要素，则相对固定要素显得太多，影响生产，边际产量减少，降低了总产量[7]。

#### **第三节 短期成本**

1. **经济分析中的各类成本**
   * **机会成本、会计成本与历史成本**[7]：
     * **历史成本**：已经发生的会计成本或帐面成本[7]。
     * **会计成本**：经济个体直接发生的一切费用和代价，在会计帐目上可以体现出来[7]。
   * **显性成本与隐性成本**[7]：
     * **显性成本**：企业直接的货币投入[7]。
     * **隐性成本**：企业投入生产的自有要素的货币价值[7]。

2. **各类短期成本曲线**
   * **短期成本**：只能对部分要素进行调整，而不能对全部要素进行调整的时期内所发生的成本[7]。
   * **固定成本与可变成本**[7]：
     * **固定成本（FC）**：不随着产量变动而变动的成本[7]。
     * **可变成本（VC）**：随着产量变动而变动的成本[7]。
   * **总成本曲线**[8]：
     * $TC = TFC + TVC = rK + wL$（r：租金，w：工资都是常数）[8]。
     * **总成本（TC）**：生产某产品的全部成本，随产量增加而增加[8]。
     * **总固定成本（TFC）**：生产某产品的所有固定成本，不随产量变化而变化[8]。
     * **总变动成本（TVC）**：生产某产品的所有变动成本，随产量增加而增加[8]。
   * **平均成本曲线**[8]：
     * **平均变动成本（AVC）**：平均每生产一个单位产品所需要的变动成本。$AVC = TVC/Q$[8]。
     * **平均成本（AC）**：平均每生产一个单位产品所需要的成本。$AC = TC/Q = AFC + AVC$[8]。
     * **平均固定成本（AFC）**：平均每生产一个单位产品所需要的固定成本。$AFC = TFC/Q$[8]。
   * **边际成本曲线**[8]：
     * **边际成本（MC）**：多生产一单位产品所引起的总成本的增加[8]。$MC = \Delta TC / \Delta Q = dTC / dQ$[8]。

3. **成本曲线之间的关系**
   * **TFC与AFC曲线**：一个产量平均固定成本的大小是对应总成本线上该点与原点连线的斜率[8]。
   * **TC、TVC与AC、AVC、MC的关系**：一个产量平均成本的大小是对应总成本线上该点与原点连线的斜率。边际成本是总成本线上对应点切线斜率[8]。
   * **AC、AVC、MC的关系**[8]：
     * $MC > AC$时，$AC$随着$Q$增加而递增。
     * $MC AC$时，$AC$随$Q$增加而递减。
     * $AVC$位于$AC$下方，与$AC$的差距逐渐缩小。
     * $MC$与$AVC$、$AC$分别交于$AVC$、$AC$最小值处。

4. **成本函数与生产函数的关系**
   * 生产与成本的关联：$AVC = TVC/Q$[2]。
   * **平均产量与平均可变成本**：平均产量越高，平均可变成本越小；反之亦然[2]。
   * **边际产量与边际成本**：边际产量越高，边际成本越小；反之亦然[2]。**边际成本递增是边际产量递减的结果**[6]。

#### **第四节 长期成本**

1. **长期总成本（LTC）**
   * **概念**：指厂商在长期中，在每一个产量水平上通过选择**最优的生产规模**所能达到的最低总成本[2]。$LTC = LTC(Q)$[2]。
   * **图形**：长期中可以调整选择最优规模，以最低成本生产[2]。规模无限分割时，LTC曲线是无数条STC曲线的包络线[2]。

2. **长期平均总成本（LAC）**
   * **概念**：指厂商在长期内按产量**平均**计算的**最低**总成本[2]。$LAC = LTC(Q) / Q$[2]。
   * **图形**：长期平均成本曲线呈U形的原因与**规模经济与规模不经济**有关[6]。LAC曲线是短期AC曲线的包络线[2][4]。

3. **长期边际成本（LMC）**
   * **概念**：表示厂商在长期内增加一单位产量所引起的最低总成本的增量[9]。$LMC = \Delta LTC(Q) / \Delta Q = dLTC(Q) / dQ$[9]。
   * **与长期平均成本的关系**：在LAC曲线的最低点，有$LMC = SMC = LAC = SAC$；$LMC LAC$时，LAC下降；$LMC > LAC$时，LAC上升[9]。

#### **第五节 规模经济与规模不经济**

1. **基本概念**
   * **规模经济**：**长期平均总成本**随产量增加而**减少**的特性[4]。
   * **规模收益不变**：长期平均总成本在产量变动时保持不变[4]。
   * **规模不经济**：长期平均总成本随产量增加而**增加**的特性[4]。

2. **产生原因**
   * **规模经济的原因**：规模扩大有利于**专业化**生产，随规模扩大，管理效率的提高、要素购买、产品销售的优势、费用的分摊等也可能是规模经济产生的原因。在产量低时，规模经济更常见[4]。
   * **规模不经济的原因**：由于随规模扩大，信息传递的成本、管理效率的降低、过细分工降低的生产效率、因为意外带来巨额损失等。当产量高时，规模不经济更常见[4]。

3. **内在经济与外在经济**
   * **内在经济**：指一个厂商在生产规模扩大的时候，由自身因素所引起的平均成本下降的现象[9]。
   * **外在经济**：指厂商因自身以外的因素引起的平均成本整体降低的现象[9]。
   * **外在不经济**：厂商因自身以外的因素引起的平均成本整体上升的现象[9]。

1. **核心概念辨析**[9]：

   * 经济利润与会计利润。
   * 总、平均和边际量的区别。

2. **重要规律**[9]：
   * **边际产量递减**，**边际成本递增**。
   * 平均成本与边际成本的关系。
   * 长期与短期的区别。
