# 第八章 垄断竞争知识点总结

## 第一节 垄断竞争市场的特点及厂商需求收益曲线

### 一、垄断竞争市场的特点
1. **有许多卖方**：市场上存在大量生产相似但不同质产品的厂商[1]
2. **产品有细微差别**：各厂商的产品在质量、包装、品牌、服务等方面存在差异，但可以互相替代[1]
3. **自由进入和退出市场**：新厂商可以相对容易地进入市场，现有厂商也可以自由退出[1]
4. **价格控制能力有限**：厂商可以对价格施加有限的影响，但由于存在替代品，控制力不如垄断厂商[1]

### 二、代表性厂商的需求曲线
1. **假设其他厂商不会对该厂商的行为做出反应**[1]
2. **产品具有差别**，导致厂商需求曲线$d$向右下方倾斜[1]
3. **比垄断厂商的需求曲线更平坦**：垄断竞争厂商对市场的控制和产品有替代品，决定了提升同样价格幅度，垄断竞争厂商量的减少幅度大于垄断厂商，即后者需求曲线的弹性偏大，更平坦[1]

### 三、代表性厂商的收益曲线
1. **假设其他厂商不会对该厂商的行为做出反应**[2]
2. **平均收益曲线即需求曲线**：$AR = P$[2]
3. **边际收益曲线位于需求曲线下方**：当需求曲线为直线时，$MR$与需求曲线纵截距相等，横截距为需求曲线的一半[2]

## 第二节 垄断竞争厂商的产量价格决策

### 一、垄断竞争厂商的短期均衡
1. **决策步骤**：
   - 根据利润最大化原则$MR=MC$，决定最佳产量[2]
   - 在需求曲线上找到最佳产量对应的价格[2]
   - 利润可能大于零、小于零或等于零[2]
2. **短期均衡条件**：$MR=MC$[2]
3. **利润情况**：
   - 当$P > SAC$时，$\pi > 0$（盈利）[2]
   - 当$P = SAC$时，$\pi = 0$（收支相抵）[2]
   - 当$P < SAC$时，$\pi < 0$（亏损）[2]

### 二、垄断竞争厂商的长期均衡
1. **长期调整过程**：
   - 利润大于零时，新厂商进入[3]
   - 生产者数量增加，市场需求不变，在位者需求降低[3]
   - 需求曲线左移，边际收益曲线左移[3]
   - 产量降低，价格降低，利润有所减少[3]
2. **长期均衡条件**：
   - $MR=MC=LMC=SMC$[3]
   - $P=AR=ATC$[3]
3. **零利润均衡**：长期均衡时厂商获得零经济利润（正常利润）[3]
4. **长期均衡特点**：
   - 产量高于垄断，低于竞争产量[1]
   - 价格高于竞争而低于垄断[1]
   - 有价格加成，有过剩生产能力[1]

## 第三节 垄断竞争市场的福利分析

### 一、产量分析
1. **垄断竞争厂商产量小于有效规模**：存在过剩生产能力[3]
2. **与完全竞争比较**：
   - 完全竞争厂商：$Q_0 = \text{有效规模}$[3]
   - 垄断竞争厂商：$Q < \text{有效规模}$[3]

### 二、价格分析
1. **存在价格加成**：$P > MC$[3]
2. **与完全竞争比较**：
   - 完全竞争：$P = MC$[3]
   - 垄断竞争：$P > MC$，存在无谓损失[3]

### 三、市场效率评价
1. **零利润均衡**：$P = AC$[3]
2. **效率比较**：
   - 效率高于垄断市场[3]
   - 效率低于完全竞争市场[3]
   - 存在无谓损失[3]
3. **综合评估**：
   - $Q < \text{有效规模}$，生产能力过剩[3]
   - $P > MC$，价格加成[3]
   - 零利润均衡：$P = AC$[3]

## 第四节 广告

### 一、使用广告的目的
1. **产品有差异**：告知消费者"差异"[3]
2. **价格高于边际成本**：增加销量则利润增加[3]
3. **思考问题**：你认为厂商做广告的目的是什么？[3]

### 二、关于广告的争论
1. **批评者观点**：
   - 企业用广告控制消费者的偏好[3]
   - 广告减少了竞争[3]
2. **辩护者观点**：
   - 广告向消费者提供了信息[3]
   - 使价格和产品质量的竞争更激烈[3]
3. **思考问题**：你对广告怎么看？你喜欢广告吗？[3]

## 本章关键知识点总结

1. **垄断竞争市场特点**：卖方许多；产品有差别；自由进退[1]
2. **厂商的需求收益曲线**：和垄断类似，只是平坦些[1]
3. **厂商短期均衡条件**：$MR=MC$[1]
4. **厂商长期均衡条件**：$MR=MC$，$P=ATC$[1]
5. **长期均衡特点**：产量高于垄断，低于竞争产量；价格高于竞争而低于垄断，有价格加成，有过剩生产能力[1]
6. **长期均衡分析图形**：需要掌握典型图形[1]
7. **广告的作用**：告知差异、增加销量、引发竞争与信息争议[3]
