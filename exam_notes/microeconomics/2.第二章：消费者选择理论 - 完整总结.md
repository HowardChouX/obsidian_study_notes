### **第二章：消费者选择理论 - 完整总结（公式修正版）**

---

### **2.1 预算线 ⭐⭐⭐⭐⭐**

#### **核心知识点**
1.  **定义**：在收入(I)和商品价格(Px, Py)既定条件下，消费者能购买的两种商品(X, Y)的所有最大数量组合。
2.  **预算方程**：`Px * X + Py * Y = I`
3.  **斜率**：`- (Px / Py)`。斜率的绝对值 `Px / Py` 称为**市场交换比率**，表示为获得1单位X必须放弃的Y的数量。
4.  **图形与变动**：
    - 横轴截距：`I / Px`（全部收入买X的数量）
    - 纵轴截距：`I / Py`（全部收入买Y的数量）
    - **收入变化**：预算线平行移动（收入增加向右移动，收入减少向左移动）。
    - **价格变化**：一种商品价格变化，预算线以其不变价格的轴交点为圆心旋转。

#### **典型例题**
- 收入 I = 100元，苹果价格 Px = 5元，香蕉价格 Py = 4元。
- 预算方程：`5X + 4Y = 100`
- 横轴截距：100/5 = 20（最多买20个苹果）
- 纵轴截距：100/4 = 25（最多买25根香蕉）
- 斜率：-5/4（每多买1个苹果，要少买1.25根香蕉）

---

### **2.2 效用 ⭐⭐⭐⭐**

#### **核心知识点**
1.  **效用**：消费者从商品消费中获得的满足程度。
2.  **总效用(TU)**：消费一定数量商品获得的总满足感。
3.  **边际效用(MU)**：每增加一单位商品消费所带来的总效用的增量。`MU = ΔTU / ΔQ`
4.  **边际效用递减规律**：在其他条件不变时，随着某种商品消费数量的连续增加，消费者从中获得的边际效用是递减的。

---

### **2.3 无差异曲线 ⭐⭐⭐⭐⭐**

#### **核心知识点**
1.  **定义**：给消费者带来相同效用水平的两种商品所有不同数量组合点的轨迹。
2.  **特征**：
    - 向右下方倾斜（为保持效用不变，增加X必须减少Y）。
    - 凸向原点（由**边际替代率递减**规律决定）。
    - 离原点越远的无差异曲线代表的效用水平越高。
    - 任意两条无差异曲线不相交。
3.  **边际替代率(MRS)**：无差异曲线上某点的斜率绝对值。`MRS_xy = - (ΔY / ΔX) = MU_x / MU_y`
    - **经济含义**：消费者在效用不变时，愿意用Y商品替代X商品的比率。

#### **典型例题**
- 如果消费者认为1个苹果和2根香蕉带来的满足感相同，那么在这一点上，苹果对香蕉的边际替代率 `MRS_xy = 2`。

---

### **2.4 消费者均衡 ⭐⭐⭐⭐⭐**

#### **核心知识点**
1.  **均衡点**：**预算线与尽可能高的无差异曲线相切的切点**。该点实现了在既定预算约束下的效用最大化。
2.  **均衡条件**：
    - **图示条件**：预算线的斜率 = 无差异曲线的斜率。
    - **数学条件**：`MU_x / P_x = MU_y / P_y`（边际效用均等法则）。
    - 将数学条件变形可得：`MU_x / MU_y = P_x / P_y`，即 `MRS_xy = P_x / P_y`。
3.  **经济含义**：消费者花费在每一种商品上的最后一元钱所带来的边际效用相等时，总效用最大。

#### **典型例题**
- 若 `MU_x / P_x > MU_y / P_y`，说明最后一元买X比买Y划算，理性消费者会增加X的消费，减少Y的消费。随着X消费增加，MU_x递减；Y消费减少，MU_y递增，直至两者相等，达到均衡。

---

### **2.5 价格变动的替代效应和收入效应 ⭐⭐⭐⭐⭐**

#### **核心知识点**
1.  **总效应**：商品价格变化引起的需求量总变化。**总效应 = 替代效应 + 收入效应**。
2.  **替代效应**：由于商品相对价格变化，消费者增加相对便宜商品的消费、减少相对昂贵商品的消费的倾向。**替代效应总是负的**（价格降，消费增）。
3.  **收入效应**：由于价格变化导致消费者实际购买力变化，进而引起的需求量变化。
    - 对于**正常商品**：收入效应为负（价格降，实际收入升，需求增）。
    - 对于**低档商品**：收入效应为正（价格降，实际收入升，需求反而减）。
4.  **希克斯分解法**：通过作一条与**新预算线平行**且与**原无差异曲线相切**的“补偿预算线”来分离两种效应。

#### **商品分类**
- **正常商品**：替代效应和收入效应均与价格反向变动，需求曲线右下倾斜。
- **低档商品**：替代效应为负，收入效应为正（但小于替代效应），总效应为负，需求曲线仍右下倾斜。
- **吉芬商品**（特殊低档品）：替代效应为负，收入效应为正**且绝对值大于替代效应**，导致总效应为正（价格降，需求减），需求曲线右上倾斜。

---

### **2.6 消费者选择理论的应用 ⭐⭐⭐⭐**

#### **核心应用**
1.  **劳动供给决策**：工资变动的替代效应（工资升，增加劳动）和收入效应（工资升，减少劳动享受闲暇）共同决定向后弯曲的劳动供给曲线。
2.  **跨期消费选择**：在现在消费与未来消费之间进行权衡，利率是相对价格。
3.  **不确定性下的选择**：分析风险偏好（风险厌恶、风险中性、风险喜好）。
