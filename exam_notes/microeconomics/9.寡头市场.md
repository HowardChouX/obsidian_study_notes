# 第九章 寡头市场知识点总结

## 第一节 寡头市场特点

### 一、寡头市场特征
1. **只有几个卖者**：市场上只有少数几家厂商[1]
2. **提供类似或者相同的产品**：各厂商提供的产品相似或互相替代[1]
3. **厂商相互依存**：每个厂商对市场都有很大影响力，却又不能单独控制市场[1]

### 二、寡头市场形成原因
1. **市场规模小**：只能容纳下几家厂商[1]
2. **政府特许与扶持**：政府给予特定企业排他性权利或支持[1]
3. **规模经济**：大规模生产带来的成本优势[1]

## 第二节 寡头决策特点

### 一、寡头市场决策特点
1. **相互依存**：
   - 利润不仅仅取决于它生产多少，还取决于其它企业生产多少[1]
   - 在做出生产决策时，每一家企业都要考虑它的决策会如何影响所有其他企业的生产决策[1]

2. **合作与利己之间的冲突**：
   - 根据是否合作，分为勾结行为的（即合作的）和独立行动的（即不合作的）[1]
   - 没有一个寡头市场模型可以对寡头市场的价格和产量决定作出一般的理论总结（古诺模型、斯威齐模型等）[1]

### 二、双寡头模型示例
1. **厂商的选择之一：勾结，形成卡特尔**：
   - 勾结：一个市场上的企业之间就产量或收取的价格达成协议[4]
   - 卡特尔：联合起来行事的企业集团，像一个垄断厂商一样决策[4]
   - 两个厂商就各自生产垄断产量的一半达成协议：每个企业：Q = 30, P = $40, 利润 = $900[4]
   - 但每个厂商都有动机违反协议，寡头厂商形成卡特尔和执行协议是困难的[3]

2. **厂商的选择之二：各自生产，形成纳什均衡**：
   - 纳什均衡：相互作用的经济主体，在假定所有其他主体所选策略为既定情况下，选择自己最优策略的状态[3]
   - 双头例子的纳什均衡是每个企业生产Q = 40[3]
   - 如果B生产Q = 40，A最好的办法是生产Q = 40；如果A生产Q = 40，B最好的办法是生产Q = 40[3]

## 第三节 市场结果的比较

### 一、产量比较
- **双寡头产量**：寡头产量大于垄断产量，小于竞争产量[2]
- **有效率的竞争结果**：P = MC = $10，Q = 120，利润 = $0[2]
- **垄断结果**：P = $40，Q = 60，利润 = $1,800[4]

### 二、价格比较
- **双寡头价格**：寡头价格高于竞争价格，但低于垄断价格[2]

### 三、福利比较
- 从产量和价格比较，效率处于垄断和竞争之间[2]

### 四、习题分析
1. 如果一个寡头行业把自己组成一个合作性的卡特尔，它生产的产量将**低于竞争水平，并等于垄断水平**（C）[2]
2. 如果一个寡头行业不进行合作，而且每个企业都自己选择自己的产量，那么，这个行业生产的产量将**高于竞争水平，并低于垄断水平**（B）[2]

## 第四节 博弈论概述

### 一、博弈论的基本概念
1. **博弈**：在策略性环境中进行策略性决策和采取策略性行动[2]
   - 策略性决策和策略性行动：每个人都要根据其他人的可能反应来决定自己的决策和行动[2]
   - 非策略性决策和行动：决策和行动时无须考虑这些决策和行动对其他人的影响以及由此而引起的其他人的反应[2]

2. **博弈论**：又称对策论，研究在策略状况下人们如何行为的理论[2]
   - 1838年的Cournot（古诺）模型，20世纪20年代法国的Borel（波雷尔）用最佳策略分析了下棋等问题[2]
   - 1944年Neumann（诺依曼）和Morgensten（摩根斯坦）合著的《博弈论和经济行为》标志了这个学科的成形[2]
   - 上个世纪50-60年代是博弈论发展的最重要时间[2]
   - 1994年Nash（纳什）、Harsanyi（海萨尼）和Selten（塞尔顿）共同获得诺贝尔经济学奖[2]

### 二、博弈的基本要素
1. **决策主体（Player）**：又称为参与人或局中人，以利润最大化为目标[5]
2. **策略（Strategy）**：是参与人为实现其目标而采取的一系列行动或行动计划，它规定在何种情况下采取何种行动[5]
3. **结果或支付（Payoff）**：表示博弈可能发生的结局，对结果的评价标准是支付函数，它可以用效用来衡量，也可以用货币报酬来表示[5]
4. **参与人、策略集和支付**构成了一个基本的博弈[5]

### 三、博弈分类
1. **参与人之间能否达成一个有约束力的协议进行决策**：合作博弈与非合作博弈（零和游戏）[5]
2. **参与人的数目**：二人博弈和多人博弈[5]
3. **各参与人的策略决定是否同时**：静态博弈和动态博弈[5]
4. **各参与人是否都清楚各种对局情况下每个参与人的信息**：完全信息博弈和不完全信息博弈[5]
5. **参与人策略空间的特征**：有限博弈和无限博弈[5]

### 四、经典博弈
1. **囚徒困境（完全信息博弈）**：
   - 利己的逻辑占据主导，体现了合作的困难[5]
   - 囚徒A坦白时，囚徒B的选择是坦白；囚徒A沉默时，囚徒B的选择是坦白。同理，二者的最优选择都是坦白[5]
   - 例1：商家价格战，"降价—降价"是均衡结果[5]

2. **智猪博弈**：
   - 因为无论大猪作何选择，小猪总会选择"等待"；大猪知道小猪会选择"等待"，从而自己选择"按"[5]

### 五、博弈均衡
1. **博弈均衡**：博弈中的所有参与者都不想改变自己策略的一种相对静止的状态[1]
2. **占优均衡**：由博弈中所有参与者的占优策略组合所构成的均衡[1]
   - 无论其他参与人采取什么策略，某参与者的唯一的最优策略就是他的占优策略[1]
   - 囚徒困境实现占优均衡[1]
3. **纳什均衡**：每个参与人的策略都是给定其他参与人的策略情况下的最佳反应[1]
   - 占优策略均衡必定是纳什均衡，而纳什均衡不一定是占优策略均衡[1]
   - 智猪博弈结果是纳什均衡[1]

## 本章关键知识点总结

1. **寡头市场特征**：少数；相似或互相替代的产品；相互依存[1]
2. **寡头决策特点**：相互依存，合作与利己的冲突[1]
3. **双寡头（不合作）市场结果**：价格低于垄断，产量高于垄断；价格高于竞争，产量低于竞争。合作时相当于垄断市场[1]
4. **博弈论**是研究策略状况下行为的理论，博弈基本要素：参与人、策略、支付[1]
5. **占优策略、纳什均衡与占优均衡**的概念[1]
6. **囚徒困境**揭示合作与利己的矛盾[1]
