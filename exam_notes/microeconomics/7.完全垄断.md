# 第七章 完全垄断知识点

## 第一节 垄断及其产生原因

### 一、垄断市场的特征
1. **一个卖者**：市场上只有一家生产者[6]
2. **产品没有相似的替代品**：垄断厂商的产品在市场上没有相近的替代品[2]
3. **新厂商难以进入**：存在进入障碍，新厂商难以进入该市场[2]
4. **推论**：厂商对于价格有一定的控制权，是价格制定者[4]

### 二、垄断产生的原因
1. **进入障碍**（barriers to entry）是垄断产生的主要原因[5]
2. **进入障碍的三种类型**：
   - **资源垄断**：生产者拥有关键性的资源，包括原材料、技术、流通渠道等[5]
   - **政府垄断**：政府给予一个人或一个企业排他性地出售某种物品或劳务的权利[2]
   - **自然垄断**：一个企业能以低于两个或更多企业的成本为整个市场供给物品或劳务[2]

## 第二节 厂商的需求收益曲线

### 一、厂商的需求曲线
1. **市场需求曲线**：垄断市场只有一个厂商，决定了垄断厂商的需求曲线就是市场需求曲线[2]
2. **曲线特征**：垄断厂商的需求曲线是一条向右下方倾斜的曲线[2]
3. **价格控制能力**：厂商可通过数量的调整来影响价格，但这也限制了垄断者由其市场势力获得利润的能力[2]

### 二、厂商的收益曲线
1. **收益变量**：
   - 总收益 $TR = P \times Q$[2]
   - 平均收益 $AR = TR/Q = P$[2]
   - 边际收益 $MR = \Delta TR/\Delta Q$[2]
2. **曲线特点**：
   - $TR$ 先增后减[1]
   - $AR=P$，右下方倾斜，随$Q$增加递减[1]
   - $MR \leq P$ 且随$Q$增加而递减[1]
3. **曲线关系**：
   - 当$MR=0$时，$TR$达到极值[1]
   - $AR$与$MR$在纵轴上的截距相等，$MR$的横轴截距是$AR$横轴截距的一半[1]

### 三、垄断企业的$MR$、$P$与$E_d$的关系
1. 当$E_d > 1$时，$MR > 0$，销售总收益与销售量同方向变动[1]
2. 当$E_d = 1$时，$MR = 0$，销售总收益达到最大值[1]
3. 当$E_d < 1$时，$MR < 0$，销售总收益与销售量反方向变动[1]

## 第三节 垄断厂商的短期均衡

### 一、垄断厂商的产量价格决策
1. **决策步骤**：
   - 根据利润最大化原则$MR=MC$，$MR$与$SMC$交点产量是$Q_0$[1]
   - 在需求曲线上找到产量$Q_0$对应的价格$P_0$[1]
   - 短期生产$Q_0$，以$P_0$出售（$P>AVC$）[1]

### 二、短期均衡状态
**均衡条件**：$MR=MC$[1]

1. **第一种情况**：$P > SAC$，$\pi = (P-AC)\times Q > 0$（盈利）[1]
2. **第二种情况**：$P = SAC$，$\pi = (P-AC)\times Q = 0$（收支相抵）[3]
3. **第三种情况**：$AVC \leq P < SAC$，$\pi = (P-AC)\times Q < 0$（亏损但继续生产）[3]

### 三、停止营业决策
- 当$P < AVC$时，$\pi = (P-AC)\times Q < 0$，停止营业[3]
- 企业的短期均衡条件适用于各类市场结构中的企业[3]

### 四、短期供给曲线
- 垄断市场不存在规律性的供给曲线[3]
- 同一价格可能对应多种产量，同一产量可能对应多种价格[3]

## 第四节 垄断厂商的长期均衡

### 一、垄断厂商生产规模的调整
- 长期是要素投入都可以变化的时期[3]
- 厂商会通过选择最优的生产规模，实现在每一个产量水平上的最低平均成本[3]
- 垄断厂商可以得到比短期更好（至少不会比短期差）的结果[3]

### 二、垄断厂商的长期均衡
1. **均衡条件**：$MR=LMC=SMC$[6]
2. **利润情况**：$\pi = (P - LAC)Q \geq 0$，价格通常会高于长期平均成本，厂商利润通常大于零[6]
3. **市场进入**：新厂商无法进入，厂商长期可以保持超额利润[6]

### 三、长期均衡图
- 长期均衡条件：$MR=LMC=SMC$[6]
- 通过规模调整，利润会增加[6]

## 第五节 垄断的福利分析

### 一、福利最大化
- 福利最大化时的产量即效率产量[7]
- 完全竞争市场均衡时总剩余最大，此时价格=边际成本[7]

### 二、垄断的福利分析
1. **产量比较**：垄断产量小于效率产量[7]
2. **价格比较**：垄断价格高于效率价格[7]
3. **福利损失**：产生无谓损失，垄断无效率[7]
4. **根本原因**：与竞争市场相比，垄断市场不能有效地配置资源，垄断者生产的产量小于社会合意的产量，且收取的价格高于边际成本[5]

## 第六节 价格歧视

### 一、价格歧视概念
- 厂商以不同价格向不同顾客出售同一种物品的经营做法[4]

### 二、价格歧视实施的条件
1. 厂商处于非完全竞争市场中[4]
2. 该物品的消费者偏好（支付意愿）不同且能够被区分开[4]
3. 各个市场是相互分离的，无法转卖或转卖的成本太高从而不值得转卖[4]

### 三、价格歧视的类型
1. **一级价格歧视**（完全价格歧视）：
   - 每单位制定不同价格，需要完全信息[4]
   - 产量达到社会最优水平，厂商获得全部市场福利，消费者剩余=0[4]
2. **二级价格歧视**：
   - 按购买量的不同，制定不同价格[4]
   - 厂商比单一价格获得更多利润，部分消费者剩余转化为生产者剩余[4]
3. **三级价格歧视**：
   - 根据市场的不同，制定不同价格[4]
   - 原则：$MR_1 = MR_2 = MC$[4]
   - 各子市场的价格与各自的$E_d$反向变动[4]

## 第七节 政府针对垄断的政策

### 一、政府政策的主要方向
与竞争市场相比，垄断市场不能有效地配置资源，垄断者生产的产量小于社会合意的产量，且收取的价格高于边际成本[5]。政府的政策主要有：
1. 努力使垄断行业更有竞争性[5]
2. 管制垄断者的行为[5]
3. 把一些私人垄断企业变为公共企业[5]
4. 不干涉，不作为[5]

### 二、具体政策案例
1. **努力使垄断行业更有竞争性**：
   - 中国电信的拆分重组：政府把原来的中国电信"一分为多"，拆出中国移动做手机业务，随后又组建了中国联通、中国铁通等[5]
2. **管制垄断行为**：
   - 国家电网、铁路网络的自然垄断：电价由国家发改委审核批准；高铁票价受政府管制，尤其是关系到民生的普通列车票价多年维持极低水平[5]
3. **把私人垄断企业变为公共企业**：
   - 中国烟草总公司：实行完全专卖，从生产到销售全部国有垄断，目的不是为了赚最多的钱，而是控制总量、保证质量、获取财政收入用于公共事业[5]
4. **不干涉不作为**（特定时期）：
   - 在21世纪初，中国的互联网公司开始崛起，政府采取了相对包容和审慎的态度，没有用传统反垄断的方式去拆分或严格限制它们[5]
   - 但当这些平台变得无比强大，出现了"二选一"、大数据杀熟、挤压小商家等问题时，政府的态度就从"审慎观察"转向了"加强监管"[2]

### 三、反垄断法律依据
- 《中华人民共和国反垄断法》2008年8月1日实施[5]
- 规定的垄断行为包括：经营者达成垄断协议；经营者滥用市场支配地位；具有或者可能具有排除、限制竞争效果的经营者集中[5]

## 本章关键知识点总结

1. **垄断特征**：唯一；无替代品；新厂商难以进入[2]
2. **进入障碍类型**（垄断产生的原因）：资源、政府、自然垄断[2]
3. **需求曲线**：向右下方倾斜（价量反向；限制价格控制力）[2]
4. **收益曲线**：位于需求曲线下方（横纵截距关系；$P>MC$）[2]
5. **利润最大化**：先根据$MR=MC$定产量，然后根据需求曲线定价格[2]
6. **短期均衡条件**：$MR=MC$（利润可能大于零、小于零或等于零）[2]
7. **长期均衡条件**：$MR=MC$（通常利润大于零，$P>ATC$）[2]
8. **垄断有福利损失**：部分交易没有实现[2]
9. **价格歧视**：概念、实施条件和类型[2]
10. **政府针对垄断的政策**[2]
