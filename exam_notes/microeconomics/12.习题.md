# 微观经济学重要章节典型计算题及详解

## 一、供求理论：弹性计算题

### 题目1：需求弹性计算
对于消费者来说，汽车是正常的消费品，当消费者对汽车的需求价格弹性为1.3，需求的收入弹性为2.2时，试计算[4]：
（1）其他条件不变，价格下降2%对需求数量的影响；
（2）其他条件不变，收入增加5%对需求数量的影响。

**解答：**

（1）价格变动对需求量的影响：
根据需求价格弹性公式：$E_d = -\frac{\Delta Q/Q}{\Delta P/P}$
已知：$E_d = 1.3$，$\Delta P/P = -2\%$
代入公式：$1.3 = -\frac{\Delta Q/Q}{-0.02}$
解得：$\Delta Q/Q = 1.3 \times 0.02 = 0.026 = 2.6\%$

**结论**：价格下降2%会使汽车需求量增加2.6%[4]。

（2）收入变动对需求量的影响：
根据需求收入弹性公式：$E_m = \frac{\Delta Q/Q}{\Delta M/M}$
已知：$E_m = 2.2$，$\Delta M/M = 5\%$
代入公式：$2.2 = \frac{\Delta Q/Q}{0.05}$
解得：$\Delta Q/Q = 2.2 \times 0.05 = 0.11 = 11\%$

**结论**：收入增加5%会使汽车需求量增加11%[4]。

## 二、消费者选择：效用最大化计算题

### 题目2：消费者均衡计算
假定某消费者的效用函数为$U=X^2Y$，商品X的价格$P_x=3$元，商品Y的价格$P_y=4$元，该消费者的收入为600元。计算该消费者对X、Y商品的最优购买量？[6]

**解答：**

**步骤1：求边际效用**
已知效用函数：$U=X^2Y$
对X求偏导得：$MU_x = \frac{\partial U}{\partial X} = 2XY$
对Y求偏导得：$MU_y = \frac{\partial U}{\partial Y} = X^2$

**步骤2：应用消费者均衡条件**
消费者均衡条件：$\frac{MU_x}{P_x} = \frac{MU_y}{P_y}$
代入得：$\frac{2XY}{3} = \frac{X^2}{4}$

**步骤3：化简关系式**
交叉相乘：$4 \times 2XY = 3 \times X^2$
$8XY = 3X^2$
两边同时除以X（假设X≠0）：$8Y = 3X$
得：$Y = \frac{3}{8}X$ ①

**步骤4：应用预算约束**
预算线方程：$P_xX + P_yY = M$
代入数值：$3X + 4Y = 600$ ②

**步骤5：联立求解**
将①式代入②式：$3X + 4 \times \frac{3}{8}X = 600$
$3X + \frac{3}{2}X = 600$
$\frac{6}{2}X + \frac{3}{2}X = 600$
$\frac{9}{2}X = 600$
$X = 600 \times \frac{2}{9} = \frac{1200}{9} = \frac{400}{3} \approx 133.33$

将X值代入①式：$Y = \frac{3}{8} \times \frac{400}{3} = \frac{400}{8} = 50$

**结论**：该消费者对X商品的最优购买量为$\frac{400}{3}$单位（约133.33单位），对Y商品的最优购买量为50单位[6]。

## 三、生产与成本：经济利润计算题

### 题目3：经济利润与会计利润计算
沙僧开出租，积攒了10万元钱，他用这些钱装修了自己靠街的门面，买了原料，开了一家年糕店。一年下来，沙僧卖年糕收益25万，前期装修费、经营中花费原料支出、水电、广告支出等费用共18万。沙僧以前开出租，年收入3万。银行年利率为3.6%。门面出租大概可得租金3.6万。
计算他的会计利润、经济利润分别是多少？[3]

**解答：**

**已知数据**：
- 总收益：25万元
- 显性成本（会计成本）：18万元
- 以前开出租的年收入（隐性成本1）：3万元
- 自有资金10万元的机会成本（隐性成本2）：$10 \times 3.6\% = 0.36$万元
- 门面出租租金（隐性成本3）：3.6万元

**步骤1：计算会计利润**
会计利润 = 总收益 - 显性成本
$= 25 - 18 = 7$万元

**步骤2：计算经济利润**
经济利润 = 总收益 - 机会成本（显性成本 + 隐性成本）
隐性成本 = 放弃的开出租收入 + 资金机会成本 + 放弃的租金收入
$= 3 + 0.36 + 3.6 = 6.96$万元

经济利润 = $25 - 18 - 6.96 = 0.04$万元 = 400元

**结论**：
- 会计利润 = 7万元
- 经济利润 = 0.04万元（400元）[3]

**经济意义**：虽然会计利润为7万元，但考虑到沙僧放弃的其他机会（开出租、资金利息、门面租金），他的经济利润仅为400元，几乎为零经济利润，这与完全竞争市场的长期均衡状态一致。

## 四、完全竞争市场：利润最大化计算题

### 题目4：完全竞争厂商利润最大化
一竞争性企业的成本函数为$TC=Q^3-9Q^2+81Q$，商品的价格为66，求这家企业利润最大化时该生产多少单位产品？[7]

**解答：**

**步骤1：确定利润最大化条件**
在完全竞争市场中，厂商利润最大化条件为：$P = MC$
已知：$P = 66$

**步骤2：求边际成本函数**
已知总成本函数：$TC = Q^3 - 9Q^2 + 81Q$
边际成本是总成本对产量的导数：
$MC = \frac{dTC}{dQ} = 3Q^2 - 18Q + 81$

**步骤3：应用利润最大化条件**
令$P = MC$：
$66 = 3Q^2 - 18Q + 81$

**步骤4：整理方程**
$3Q^2 - 18Q + 81 - 66 = 0$
$3Q^2 - 18Q + 15 = 0$

**步骤5：解二次方程**
两边同时除以3：$Q^2 - 6Q + 5 = 0$
因式分解：$(Q-1)(Q-5) = 0$
解得：$Q_1 = 1$，$Q_2 = 5$

**步骤6：验证利润情况**
需要检查哪个产量水平下利润大于零：
当$Q=1$时：$TC = 1^3 - 9\times1^2 + 81\times1 = 1 - 9 + 81 = 73$
$TR = P \times Q = 66 \times 1 = 66$
利润 = $TR - TC = 66 - 73 = -7$（亏损）

当$Q=5$时：$TC = 5^3 - 9\times5^2 + 81\times5 = 125 - 225 + 405 = 305$
$TR = 66 \times 5 = 330$
利润 = $330 - 305 = 25$（盈利）

**结论**：企业利润最大化时应生产5单位产品[7]。

## 五、要素市场：厂商使用要素原则计算题

### 题目5：完全竞争厂商要素使用决策
若企业所处产品与要素市场均为完全竞争，已经实现利润最大化，工人工资率为5，边际产量为0.5，则产品价格为（ ）
A. 10  B. 2.5  C. 0.1  D. 0.5

**解答：**

**步骤1：确定完全竞争市场要素使用原则**
在完全竞争市场中，厂商使用生产要素的原则为：$VMP = W$
其中：$VMP = MP \times P$（边际产品价值 = 边际产量 × 产品价格）
$W$为要素价格（工资率）

**步骤2：代入已知数据**
已知：$W = 5$，$MP = 0.5$
根据$VMP = W$：$MP \times P = W$
$0.5 \times P = 5$

**步骤3：求解产品价格**
$P = \frac{5}{0.5} = 10$

**结论**：产品价格为10，正确答案为A[1]。

## 六、市场失灵：外部性计算题

### 题目6：科斯定理应用
一家化肥厂排放的废气使附近10个农场的果园产量减产，每户损失1000元，总损失1万元。存在两种应对污染的技术方案：
1. 化肥厂烟囱上安装治理废气装置，费用6000元
2. 让果农生产蔬菜，废气对蔬菜生产无影响，转产将使每个果农发生800元调节损失，总共损失8000元

根据科斯定理分析不同产权安排下的解决方案。

**解答：**

**步骤1：确定有效解决方案**
比较两种方案的成本：
- 方案1成本：6000元
- 方案2成本：8000元
方案1成本较低，是有效解决方案。

**步骤2：分析不同产权安排**
（1）如果给化肥厂排废气权利：
果农面临选择：承受10000元损失或集资安装治理装置
果农会选择集资6000元让化肥厂安装治理废气装置，因为6000元 < 10000元

（2）如果给果农免受废气污染的权利：
化肥厂面临选择：安装治理装置（6000元）或赔偿果农损失（10000元）
化肥厂会选择投资6000元安装治理废气装置，因为6000元 < 10000元

**结论**：无论产权如何分配（给化肥厂排污权或给果农免受污染权），最终都会实现有效解决方案——安装治理装置，成本为6000元。这验证了科斯定理：只要产权明确且交易成本为零或很小，无论初始产权如何分配，最终都能达到资源配置有效率状态[1]。

## 计算题总结要点

1. **弹性计算**：掌握需求价格弹性、收入弹性的定义式和计算方法，注意弹性系数与变动方向的关系[4]。

2. **消费者均衡**：掌握效用函数求偏导得边际效用，应用均衡条件$\frac{MU_1}{P_1} = \frac{MU_2}{P_2}$，结合预算约束求解[6]。

3. **利润计算**：区分会计利润（总收益-显性成本）和经济利润（总收益-机会成本），机会成本包括显性成本和隐性成本[3]。

4. **完全竞争厂商决策**：短期利润最大化条件$P=MC$，长期均衡条件$P=LAC$，注意验证利润是否大于零[7]。

5. **要素市场**：完全竞争市场要素使用原则$VMP=W$，其中$VMP=MP×P$[1]。

6. **科斯定理应用**：明确产权分配不影响最终效率结果，只要交易成本足够低，市场能自发解决外部性问题[1]。

这些计算题涵盖了微观经济学的核心章节和重要概念，通过练习可以加深对理论的理解和应用能力。
