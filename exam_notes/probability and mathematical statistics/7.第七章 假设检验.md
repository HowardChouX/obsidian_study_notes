### ** 学习建议**
本章是数理统计的另一核心应用，与参数估计相辅相成，旨在利用样本信息对关于总体分布的某种“假设”做出接受或拒绝的统计决策。它建立了基于概率反证法的统计推断逻辑。
- **核心重点（★★★★★ 和 ★★★★☆）**：7.1, 7.2, 7.3 节是重中之重。必须深刻理解假设检验的基本思想和步骤，并熟练掌握正态总体下关于均值、方差的各种检验方法（U检验、t检验、χ²检验、F检验）。
- **理解关键（★★★☆☆）**：7.4, 7.5 节是核心方法在非标准情况下的推广和应用，需要理解其基本思想。
- **核心思路**：本章遵循“提出假设 -> 构造检验统计量 -> 确定拒绝域 -> 根据样本决策”的基本流程，核心是掌握不同情境下检验统计量的选择及其分布。

### 7.1 假设检验的基本概念 ★★★★★
- **基本思想**：基于**小概率原理**（小概率事件在一次试验中几乎不可能发生）的**概率反证法**。为了检验一个假设（原假设 $H_0$）是否成立，先假定 $H_0$ 成立，然后看在此前提下，样本观测结果出现的概率。如果这个概率很小（小于显著性水平 $\alpha$），则怀疑原假设不真，从而拒绝 $H_0$。
- **基本步骤**：
    1. **提出原假设 $H_0$ 与备择假设 $H_1$**：$H_0$ 通常是需要检验的、保守的、期望被否定的假设；$H_1$ 是与 $H_0$ 对立的、研究者希望证实的假设。
    2. **构造检验统计量**：选取一个合适的样本函数（统计量），要求其分布在 $H_0$ 为真时已知或可确定。
    3. **确定显著性水平 $\alpha$ 和拒绝域**：给定一个小概率 $\alpha$（通常为 0.05, 0.01 等），根据检验统计量的分布和 $H_1$ 的形式，确定一个区域（拒绝域 $W$），使得当 $H_0$ 为真时，统计量落入 $W$ 的概率为 $\alpha$，即 $P\{(X_1,...,X_n) \in W | H_0\} = \alpha$。
    4. **作出判断**：根据样本观测值计算检验统计量的值。若该值落入拒绝域 $W$，则**拒绝 $H_0$**，接受 $H_1$；否则**不拒绝（接受）$H_0$**。
- **两类错误**：
    - **第一类错误（弃真错误）**：$H_0$ 为真时，拒绝了 $H_0$。其概率记为 $\alpha$，即显著性水平。
    - **第二类错误（取伪错误）**：$H_0$ 不真时，接受了 $H_0$。其概率记为 $\beta$。
    - **关系**：在样本量固定时，$\alpha$ 减小会导致 $\beta$ 增大，反之亦然。

### 7.2 正态总体参数的假设检验 ★★★★★
- **核心**：利用第五章的抽样分布定理构造检验统计量。
- **单正态总体均值 $\mu$ 的检验**：
    - **$Z$ 检验（$U$ 检验）**：适用于 $\sigma^2$ **已知**。
        - 检验统计量：$Z = \frac{\bar{X} - \mu_0}{\sigma / \sqrt{n}} \sim N(0,1)$ （在 $H_0: \mu = \mu_0$ 成立时）。
        - 拒绝域形式取决于 $H_1$：
            - $H_1: \mu > \mu_0$，拒绝域：$\{Z > z_\alpha\}$
            - $H_1: \mu < \mu_0$，拒绝域：$\{Z < -z_\alpha\}$
            - $H_1: \mu \neq \mu_0$，拒绝域：$\{|Z| > z_{\alpha/2}\}$
    - **$t$ 检验**：适用于 $\sigma^2$ **未知**。
        - 检验统计量：$T = \frac{\bar{X} - \mu_0}{S / \sqrt{n}} \sim t(n-1)$ （在 $H_0: \mu = \mu_0$ 成立时）。
        - 拒绝域形式与 $Z$ 检验类似，将 $z$ 分位数换为 $t$ 分位数即可。
- **单正态总体方差 $\sigma^2$ 的检验（$\chi^2$ 检验）**：
    - 通常假设 $\mu$ **未知**。
        - 检验统计量：$\chi^2 = \frac{(n-1)S^2}{\sigma_0^2} \sim \chi^2(n-1)$ （在 $H_0: \sigma^2 = \sigma_0^2$ 成立时）。
        - 拒绝域形式取决于 $H_1$：
            - $H_1: \sigma^2 > \sigma_0^2$，拒绝域：$\{\chi^2 > \chi_\alpha^2(n-1)\}$
            - $H_1: \sigma^2 < \sigma_0^2$，拒绝域：$\{\chi^2 < \chi_{1-\alpha}^2(n-1)\}$
            - $H_1: \sigma^2 \neq \sigma_0^2$，拒绝域：$\{\chi^2 > \chi_{\alpha/2}^2(n-1) \quad \text{或} \quad \chi^2 < \chi_{1-\alpha/2}^2(n-1)\}$

### 7.3 两个正态总体参数的假设检验 ★★★★☆
- **两个总体均值差 $\mu_1 - \mu_2$ 的检验**：
    - **$Z$ 检验**：两总体方差 $\sigma_1^2, \sigma_2^2$ **已知**。
        - 检验统计量：$Z = \frac{(\bar{X} - \bar{Y}) - (\mu_1 - \mu_2)}{\sqrt{\frac{\sigma_1^2}{n_1} + \frac{\sigma_2^2}{n_2}}} \sim N(0,1)$ （在 $H_0: \mu_1 - \mu_2 = \delta_0$（常为0）成立时）。
    - **$t$ 检验**：两总体方差 $\sigma_1^2 = \sigma_2^2$ **但未知**。
        - 检验统计量：$T = \frac{(\bar{X} - \bar{Y}) - \delta_0}{S_w \sqrt{\frac{1}{n_1} + \frac{1}{n_2}}} \sim t(n_1+n_2-2)$，其中 $S_w^2 = \frac{(n_1-1)S_1^2 + (n_2-1)S_2^2}{n_1+n_2-2}$。
- **两个总体方差比 $\sigma_1^2 / \sigma_2^2$ 的检验（$F$ 检验）**：
    - 通常假设 $\mu_1, \mu_2$ **未知**。
        - 检验统计量：$F = \frac{S_1^2 / \sigma_1^2}{S_2^2 / \sigma_2^2} \sim F(n_1-1, n_2-1)$ （在 $H_0: \sigma_1^2 / \sigma_2^2 = 1$ 成立时，即 $\sigma_1^2 = \sigma_2^2$）。
        - 为检验 $H_0: \sigma_1^2 = \sigma_2^2$，统计量简化为 $F = \frac{S_1^2}{S_2^2} \sim F(n_1-1, n_2-1)$。
        - 拒绝域（以 $H_1: \sigma_1^2 \neq \sigma_2^2$ 为例）：$\{F > F_{\alpha/2}(n_1-1, n_2-1) \quad \text{或} \quad F < F_{1-\alpha/2}(n_1-1, n_2-1)\}$。通常将较大的样本方差放在分子上，只需查右侧分位数。

### 7.4 非正态总体参数的假设检验 ★★★☆☆
- **大样本方法**：根据**中心极限定理**，当样本容量 $n$ 充分大时，许多统计量近似服从正态分布。
    - **总体均值 $\mu$ 的检验**：即使总体分布未知，当 $n$ 很大时，在 $H_0: \mu = \mu_0$ 下，$Z = \frac{\bar{X} - \mu_0}{S / \sqrt{n}} \stackrel{\text{近似}}{\sim} N(0,1)$。可近似使用 $Z$ 检验。
- **注意**：此法依赖于大样本，且对于方差等参数的检验，大样本性质可能更复杂。

### 7.5 总体分布的假设检验 ★★★☆☆
- **问题**：检验总体 $X$ 的分布函数 $F(x)$ 是否等于某个给定的分布函数 $F_0(x)$，即 $H_0: F(x) = F_0(x)$。
- **$\chi^2$ 拟合优度检验**：
    - **适用**：总体分布为离散型，或连续型但已分组。
    - **步骤**：
        1. 将样本观测值划分成 $k$ 个互不相容的区间（组）。
        2. 计算样本落入每个区间的实际频数 $n_i$。
        3. 在 $H_0$ 成立下，计算理论概率 $p_i$ 和理论频数 $np_i$。
        4. 构造皮尔逊统计量：$\chi^2 = \sum_{i=1}^{k} \frac{(n_i - np_i)^2}{np_i}$。当 $n$ 很大时，该统计量近似服从自由度为 $k-r-1$ 的 $\chi^2$ 分布，其中 $r$ 是 $F_0(x)$ 中被估计的参数的个数。
        5. 给定 $\alpha$，拒绝域为 $\{\chi^2 > \chi_\alpha^2(k-r-1)\}$。
- **优点**：应用范围广，适用于任何分布。
- **缺点**：依赖于分组，会损失信息，且需要大样本。
