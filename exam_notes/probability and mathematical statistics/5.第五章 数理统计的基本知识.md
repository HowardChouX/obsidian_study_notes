### 5.1 总体与样本 ★★★★☆
- **总体**：研究对象的某项数量指标的全体。它是一个随机变量 $X$，其分布称为**总体分布**。
- **个体**：总体中的每一个元素。
- **样本**：从总体中随机抽取的 $n$ 个个体 $(X_1, X_2, ..., X_n)$，称为总体 $X$ 的一个**样本**，$n$ 称为**样本容量**。
- **简单随机样本**：满足以下两个条件的样本：
    1. **代表性**：每个 $X_i$ 与总体 $X$ 同分布。
    2. **独立性**：$X_1, X_2, ..., X_n$ 相互独立。
- **样本值**：样本的一次具体观察结果 $(x_1, x_2, ..., x_n)$，称为**样本观测值**或**样本值**。
- **重要性**：明确了统计学的核心任务——通过样本信息来推断总体的性质。

### 5.2 样本函数与统计量 ★★★★★
- **统计量**：设 $X_1, X_2, ..., X_n$ 是来自总体 $X$ 的一个样本，$g(X_1, X_2, ..., X_n)$ 是样本的函数，若 $g$ 中不含任何未知参数，则称 $g(X_1, X_2, ..., X_n)$ 为一个**统计量**。
- **常用统计量**：
    - **样本均值**：$\bar{X} = \frac{1}{n}\sum_{i=1}^{n}X_i$，反映数据的中心位置。
    - **样本方差**：$S^2 = \frac{1}{n-1}\sum_{i=1}^{n}(X_i - \bar{X})^2$，反映数据的离散程度。（注意分母是 $n-1$，这是为了满足无偏性）
    - **样本标准差**：$S = \sqrt{S^2}$。
    - **样本 $k$ 阶原点矩**：$A_k = \frac{1}{n}\sum_{i=1}^{n}X_i^k$。
    - **样本 $k$ 阶中心矩**：$B_k = \frac{1}{n}\sum_{i=1}^{n}(X_i - \bar{X})^k$。
- **重要性**：统计量是连接样本与总体的桥梁，是进行统计推断的直接工具。统计量本身也是随机变量。

### 5.3 数理统计中的某些常用分布 ★★★☆☆
- **$\chi^2$ 分布（卡方分布）**：
    - **定义**：设 $X_1, X_2, ..., X_n$ 相互独立，均服从标准正态分布 $N(0,1)$，则称随机变量 $\chi^2 = \sum_{i=1}^{n}X_i^2$ 服从自由度为 $n$ 的 $\chi^2$ 分布，记为 $\chi^2 \sim \chi^2(n)$。
    - **性质**：1. $E(\chi^2(n)) = n$, $D(\chi^2(n)) = 2n$。 2. 可加性：若 $\chi_1^2 \sim \chi^2(n_1)$, $\chi_2^2 \sim \chi^2(n_2)$，且相互独立，则 $\chi_1^2 + \chi_2^2 \sim \chi^2(n_1+n_2)$。
- **$t$ 分布（学生氏分布）**：
    - **定义**：设 $X \sim N(0,1)$, $Y \sim \chi^2(n)$，且 $X$ 与 $Y$ 相互独立，则称随机变量 $t = \frac{X}{\sqrt{Y/n}}$ 服从自由度为 $n$ 的 $t$ 分布，记为 $t \sim t(n)$。
    - **性质**：1. 密度函数关于 $t=0$ 对称。 2. 当 $n \to \infty$ 时，$t$ 分布近似于标准正态分布。
- **$F$ 分布**：
    - **定义**：设 $U \sim \chi^2(n_1)$, $V \sim \chi^2(n_2)$，且 $U$ 与 $V$ 相互独立，则称随机变量 $F = \frac{U/n_1}{V/n_2}$ 服从自由度为 $(n_1, n_2)$ 的 $F$ 分布，记为 $F \sim F(n_1, n_2)$。
    - **性质**：若 $F \sim F(n_1, n_2)$，则 $\frac{1}{F} \sim F(n_2, n_1)$。
- **重要性**：这三个分布是正态总体下导出统计量分布的基础，其分位数是构造置信区间和进行假设检验的临界值。

### 5.4 正态总体统计量的分布 ★★★★★
- **核心结论**：以下结论均在“总体 $X \sim N(\mu, \sigma^2)$，$X_1, X_2, ..., X_n$ 是简单随机样本”的假设下成立。
    1. **样本均值的分布**：$\bar{X} \sim N(\mu, \frac{\sigma^2}{n})$，或等价地，$\frac{\bar{X} - \mu}{\sigma/\sqrt{n}} \sim N(0,1)$。
    2. **样本均值与样本方差的独立性**：$\bar{X}$ 与 $S^2$ 相互独立。
    3. **与样本方差相关的分布**：$\frac{(n-1)S^2}{\sigma^2} = \frac{\sum_{i=1}^{n}(X_i - \bar{X})^2}{\sigma^2} \sim \chi^2(n-1)$。（自由度是 $n-1$）
    4. **$t$ 分布的出现**：将结论1中的未知参数 $\sigma$ 用样本标准差 $S$ 代替，则 $\frac{\bar{X} - \mu}{S/\sqrt{n}} \sim t(n-1)$。
- **两个正态总体的情形**：设 $X_1, ..., X_{n_1}$ 来自 $N(\mu_1, \sigma_1^2)$，$Y_1, ..., Y_{n_2}$ 来自 $N(\mu_2, \sigma_2^2)$，且两样本独立。
    1. **均值差的分布**：$\bar{X} - \bar{Y} \sim N(\mu_1 - \mu_2, \frac{\sigma_1^2}{n_1} + \frac{\sigma_2^2}{n_2})$。
    2. **方差比的分布**：$\frac{S_1^2 / \sigma_1^2}{S_2^2 / \sigma_2^2} \sim F(n_1-1, n_2-1)$。
- **重要性**：这些抽样分布定理是第六章（参数估计）和第七章（假设检验）几乎所有公式推导和结论成立的直接理论依据，必须牢固掌握。
