### 2.1 随机变量的概念 ★★★☆☆
- **定义**：定义在样本空间 $S$ 上的实值函数 $X = X(e), e \in S$。它将随机试验的每一个可能结果 $e$ 用一个实数 $X(e)$ 表示。
- **作用**：使随机事件的研究数量化、系统化。例如，掷骰子，将“出现1点”这个事件与数值 `1` 对应。
- **分类**：离散型随机变量和连续型随机变量。

### 2.2 离散随机变量 ★★★★★
- **定义**：全部可能取值为有限个或可数无限个的随机变量。
- **分布律（概率分布）**：描述离散随机变量 $X$ 取各个可能值 $x_k$ 的概率。
    - **表示形式**：$P\{X = x_k\} = p_k, k=1,2,...$
    - **性质**：1. $p_k \geq 0$； 2. $\sum_{k=1}^{\infty}p_k = 1$。

### 2.3 超几何分布·二项分布·泊松分布 ★★★★★
- **超几何分布**：描述$N$个产品中有$M$个次品，无放回抽取$n$个，其中次品数$X$的分布。
    - 公式：$P\{X=k\} = \frac{C_M^k C_{N-M}^{n-k}}{C_N^n}$。
- **二项分布 $X \sim B(n, p)$**：描述$n$重伯努利试验中成功次数$X$的分布。
    - 公式：$P\{X=k\} = C_n^k p^k (1-p)^{n-k}$。
- **泊松分布 $X \sim P(\lambda)$**：描述单位时间/面积内稀有事件发生次数$X$的分布。当$n$很大$p$很小($np=\lambda$)时，可近似二项分布。
    - 公式：$P\{X=k\} = \frac{\lambda^k}{k!}e^{-\lambda}$。

### 2.4 连续随机变量 ★★★★★
- **定义**：可能取值充满某个区间（不可数）的随机变量。描述其分布不能用分布律，需用**概率密度函数**。

### 2.5 随机变量的分布函数 ★★★★☆
- **定义**：对任意实数$x$，函数 $F(x) = P\{X \leq x\}$ 称为$X$的分布函数。
- **性质**：
    1. $0 \leq F(x) \leq 1$。
    2. $F(x)$是单调不减函数。
    3. $F(-\infty) = \lim_{x \to -\infty}F(x) = 0$, $F(+\infty) = \lim_{x \to +\infty}F(x) = 1$。(规范性)
    4. $F(x)$是右连续函数，即 $F(x+0) = F(x)$。
- **通用性**：对离散型和连续型随机变量均适用。

### 2.6 连续随机变量的概率密度 ★★★★★
- **定义**：若存在非负可积函数 $f(x)$，使随机变量$X$的分布函数 $F(x) = \int_{-\infty}^{x} f(t) dt$，则称 $f(x)$ 为 $X$ 的概率密度函数（PDF）。
- **性质**：
    1. $f(x) \geq 0$。
    2. $\int_{-\infty}^{+\infty} f(x) dx = 1$。
    3. $P\{x_1 < X \leq x_2\} = F(x_2) - F(x_1) = \int_{x_1}^{x_2} f(x) dx$。
    - **注意**：$f(x)$ 不是概率，$P\{X=x\}=0$。$f(x)$ 反映了概率在$x$点附近的“密集程度”。

### 2.7 均匀分布·指数分布 ★★★★☆
- **均匀分布 $X \sim U(a, b)$**：在区间 $[a, b]$ 上等可能取值的随机变量。
    - 概率密度：$f(x) = \begin{cases} \frac{1}{b-a}, & a \leq x \leq b \\ 0, & \text{其他} \end{cases}$
    - 分布函数：$F(x) = \begin{cases} 0, & x < a \\ \frac{x-a}{b-a}, & a \leq x < b \\ 1, & x \geq b \end{cases}$
- **指数分布 $X \sim Exp(\lambda)$**：常用于表示独立随机事件发生的时间间隔。
    - 概率密度：$f(x) = \begin{cases} \lambda e^{-\lambda x}, & x > 0 \\ 0, & x \leq 0 \end{cases}$
    - 分布函数：$F(x) = \begin{cases} 1 - e^{-\lambda x}, & x > 0 \\ 0, & x \leq 0 \end{cases}$
    - **无记忆性**：$P\{X > s+t | X > s\} = P\{X > t\}$。

### 2.8 随机变量的函数的分布 ★★★☆☆
- **问题**：已知随机变量$X$的分布，求$Y=g(X)$的分布。
- **方法**：
    - **公式法（适用于$g(x)$单调）**：若 $Y=g(X)$，$g(x)$ 严格单调，其反函数 $x = h(y)$ 有连续导数，则 $Y$ 的概率密度为：$f_Y(y) = f_X(h(y)) \cdot |h'(y)|$。
    - **分布函数法（通用）**：先求 $Y$ 的分布函数 $F_Y(y) = P\{Y \leq y\} = P\{g(X) \leq y\}$，再对 $y$ 求导得到概率密度 $f_Y(y)$。

### 2.9 二维随机变量的联合分布 ★★★★★
- **定义**：设 $E$ 是随机试验，样本空间 $S=\{e\}$，$X=X(e)$ 和 $Y=Y(e)$ 是定义在 $S$ 上的随机变量，由它们构成的向量 $(X, Y)$ 叫**二维随机变量**。
- **联合分布函数**：$F(x, y) = P\{X \leq x, Y \leq y\}$。
- **离散型联合分布律**：$P\{X=x_i, Y=y_j\} = p_{ij}$。
- **连续型联合概率密度**：若存在非负函数 $f(x, y)$，使 $F(x, y) = \int_{-\infty}^{y}\int_{-\infty}^{x} f(u, v) du dv$，则 $f(x, y)$ 为联合概率密度。

### 2.10 二维随机变量的边缘分布 ★★★★☆
- **定义**：二维随机变量 $(X, Y)$ 中，$X$ 和 $Y$ 各自的分布称为**边缘分布**。
- **由联合分布求边缘分布**：
    - **离散型**：$P\{X=x_i\} = \sum_{j} p_{ij}$（联合分布律对 $j$ 求和）。
    - **连续型**：$f_X(x) = \int_{-\infty}^{+\infty} f(x, y) dy$（联合概率密度对 $y$ 积分）。

### 2.11 二维随机变量的条件分布 ★★★☆☆
- **定义**：在一个随机变量取固定值的条件下，另一个随机变量的分布。
- **离散型**：$P\{X=x_i | Y=y_j\} = \frac{p_{ij}}{p_{\cdot j}}$（要求 $P\{Y=y_j\} > 0$）。
- **连续型**：在 $Y=y$ 条件下 $X$ 的**条件概率密度**为 $f_{X|Y}(x|y) = \frac{f(x, y)}{f_Y(y)}$（要求 $f_Y(y) > 0$）。

### 2.12 随机变量的独立性 ★★★★★
- **定义**：若对于所有 $x, y$，联合分布函数等于边缘分布函数的乘积，即 $F(x, y) = F_X(x)F_Y(y)$，则称 $X$ 与 $Y$ 相互独立。
- **等价条件**：
    - **离散型**：对一切 $i, j$，有 $P\{X=x_i, Y=y_j\} = P\{X=x_i\}P\{Y=y_j\}$。
    - **连续型**：在 $f(x, y)$ 的所有连续点上，有 $f(x, y) = f_X(x)f_Y(y)$。
- **重要性**：独立性是简化概率模型的关键。

### 2.13 二维随机变量的函数分布 ★★★☆☆
- **问题**：已知 $(X, Y)$ 的联合分布，求 $Z = g(X, Y)$ 的分布（如 $Z=X+Y, Z=XY, Z=max(X,Y)$）。
- **常用方法**：分布函数法。先求 $Z$ 的分布函数 $F_Z(z) = P\{Z \leq z\} = P\{g(X, Y) \leq z\} = \iint_{g(x,y) \leq z} f(x, y) dx dy$，再求导得概率密度。
- **卷积公式（$Z=X+Y$）**：若 $X$ 和 $Y$ 独立，则 $Z=X+Y$ 的概率密度为 $f_Z(z) = \int_{-\infty}^{+\infty} f_X(x)f_Y(z-x) dx$。
