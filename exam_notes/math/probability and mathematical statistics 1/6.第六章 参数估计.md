### **第六章 参数估计 - 学习建议**
本章是数理统计的核心应用章节，旨在解决如何利用样本信息对未知的总体参数进行推断。它建立了从“已知样本”到“推测总体”的完整方法论。
- **核心重点（★★★★★ 和 ★★★★☆）**：6.1, 6.2, 6.3 节是重中之重。必须熟练掌握矩估计和极大似然估计两种点估计方法，理解评价估计量好坏的标准，并学会构造正态总体参数的置信区间。
- **理解关键（★★★☆☆）**：6.4, 6.6 节是单总体情形的自然推广和补充，需要掌握其思想和方法。
- **核心思路**：本章介绍了参数估计的两种形式：**点估计**（给出一个数值）和**区间估计**（给出一个范围）。核心是掌握估计量的构造方法及其性质评价。

---
### 6.1 参数的点估计 ★★★★★
- **问题**：设总体 $X$ 的分布函数 $F(x; \theta)$ 形式已知，但参数 $\theta$ 未知。用样本 $X_1, X_2, ..., X_n$ 构造一个统计量 $\hat{\theta} = \hat{\theta}(X_1, X_2, ..., X_n)$ 来估计 $\theta$，称为**点估计**。$\hat{\theta}$ 称为**估计量**，代入样本观测值得到的值称为**估计值**。
- **矩估计法**：
    - **核心思想**：用样本矩作为总体矩的估计。
    - **方法**：令总体矩等于样本矩，建立方程求解。
        - 总体 $k$ 阶原点矩 $\nu_k = E(X^k)$。
        - 样本 $k$ 阶原点矩 $A_k = \frac{1}{n}\sum_{i=1}^{n}X_i^k$。
        - 建立方程组：$\nu_k(\theta_1, ..., \theta_m) = A_k, \quad k=1,2,...,m$，解出未知参数 $\theta_1, ..., \theta_m$ 的矩估计量。
    - **优点**：简单易行，通常具有相合性。
    - **缺点**：不唯一（可以用不同阶的矩），有时不够理想。
- **极大似然估计法**：
    - **核心思想**：“概率最大的事件最可能发生”。选择使样本观测值出现的概率达到最大的参数值作为估计值。
    - **似然函数**：
        - **离散型**：$L(\theta) = L(x_1, ..., x_n; \theta) = \prod_{i=1}^{n} P\{X_i = x_i\}$。
        - **连续型**：$L(\theta) = L(x_1, ..., x_n; \theta) = \prod_{i=1}^{n} f(x_i; \theta)$。
    - **方法**：求似然函数 $L(\theta)$ 的最大值点。通常通过求解**似然方程** $\frac{d \ln L(\theta)}{d\theta} = 0$ 得到。
    - **优点**：具有一系列优良的统计性质（如相合性、渐近正态性等），是应用最广泛的估计方法。

### 6.2 衡量点估计量好坏的标准 ★★★★☆
- **无偏性**：
    - **定义**：若估计量 $\hat{\theta}$ 的数学期望等于被估计参数的真值，即 $E(\hat{\theta}) = \theta$，则称 $\hat{\theta}$ 是 $\theta$ 的**无偏估计量**。
    - **意义**：说明估计量没有系统偏差。
    - **例子**：样本均值 $\bar{X}$ 是总体均值 $\mu$ 的无偏估计；样本方差 $S^2 = \frac{1}{n-1}\sum (X_i - \bar{X})^2$ 是总体方差 $\sigma^2$ 的无偏估计（而 $\frac{1}{n}\sum (X_i - \bar{X})^2$ 则不是）。
- **有效性**：
    - **定义**：设 $\hat{\theta}_1$ 和 $\hat{\theta}_2$ 都是 $\theta$ 的无偏估计量，若 $D(\hat{\theta}_1) < D(\hat{\theta}_2)$，则称 $\hat{\theta}_1$ 比 $\hat{\theta}_2$ **有效**。
    - **意义**：在无偏的前提下，方差越小，估计量越有效，波动越小。
- **相合性（一致性）**：
    - **定义**：若对于任意 $\epsilon > 0$，有 $\lim_{n \to \infty} P\{|\hat{\theta} - \theta| < \epsilon\} = 1$，则称 $\hat{\theta}$ 是 $\theta$ 的**相合估计量**。
    - **意义**：当样本容量无限增大时，估计量依概率收敛于参数真值。这是对估计量的基本要求。

### 6.3 正态总体参数的区间估计 ★★★★★
- **基本概念**：寻找一个随机区间 $(\hat{\theta}_1, \hat{\theta}_2)$，使得该区间以一定的概率（**置信水平** $1-\alpha$）包含未知参数 $\theta$ 的真值，即 $P\{\hat{\theta}_1 < \theta < \hat{\theta}_2\} = 1-\alpha$。这个区间称为 $\theta$ 的**置信区间**，$1-\alpha$ 称为**置信度**（通常取 0.95, 0.99 等）。
- **枢轴量法**：构造区间估计的一般步骤：
    1. 寻找一个包含待估参数 $\theta$ 和样本的函数 $G = G(X_1, ..., X_n; \theta)$，且 $G$ 的分布已知且与 $\theta$ 无关。$G$ 称为**枢轴量**。
    2. 对于给定的置信度 $1-\alpha$，确定常数 $a, b$，使得 $P\{a < G < b\} = 1-\alpha$。
    3. 将不等式 $a < G < b$ 等价变形为 $\hat{\theta}_1 < \theta < \hat{\theta}_2$，则 $(\hat{\theta}_1, \hat{\theta}_2)$ 即为所求置信区间。
- **单正态总体均值的区间估计**：
    - **$\sigma^2$ 已知**：枢轴量为 $U = \frac{\bar{X} - \mu}{\sigma / \sqrt{n}} \sim N(0,1)$。$\mu$ 的 $1-\alpha$ 置信区间为：$(\bar{X} - z_{\alpha/2} \frac{\sigma}{\sqrt{n}}, \bar{X} + z_{\alpha/2} \frac{\sigma}{\sqrt{n}})$。
    - **$\sigma^2$ 未知**：枢轴量为 $T = \frac{\bar{X} - \mu}{S / \sqrt{n}} \sim t(n-1)$。$\mu$ 的 $1-\alpha$ 置信区间为：$(\bar{X} - t_{\alpha/2}(n-1) \frac{S}{\sqrt{n}}, \bar{X} + t_{\alpha/2}(n-1) \frac{S}{\sqrt{n}})$。
- **单正态总体方差的区间估计**：
    - **$\mu$ 未知**（最常见）：枢轴量为 $\chi^2 = \frac{(n-1)S^2}{\sigma^2} \sim \chi^2(n-1)$。$\sigma^2$ 的 $1-\alpha$ 置信区间为：$(\frac{(n-1)S^2}{\chi^2_{\alpha/2}(n-1)}, \frac{(n-1)S^2}{\chi^2_{1-\alpha/2}(n-1)})$。

### 6.4 两个正态总体均值差及方差比的区间估计 ★★★☆☆
- **两个总体均值差 $\mu_1 - \mu_2$ 的区间估计**：
    - **$\sigma_1^2, \sigma_2^2$ 已知**：枢轴量为 $U = \frac{(\bar{X} - \bar{Y}) - (\mu_1 - \mu_2)}{\sqrt{\frac{\sigma_1^2}{n_1} + \frac{\sigma_2^2}{n_2}}} \sim N(0,1)$。
    - **$\sigma_1^2 = \sigma_2^2 = \sigma^2$ 但未知**：枢轴量为 $T = \frac{(\bar{X} - \bar{Y}) - (\mu_1 - \mu_2)}{S_w \sqrt{\frac{1}{n_1} + \frac{1}{n_2}}} \sim t(n_1+n_2-2)$，其中 $S_w^2 = \frac{(n_1-1)S_1^2 + (n_2-1)S_2^2}{n_1+n_2-2}$。
- **两个总体方差比 $\sigma_1^2 / \sigma_2^2$ 的区间估计**：
    - **$\mu_1, \mu_2$ 未知**：枢轴量为 $F = \frac{S_1^2 / \sigma_1^2}{S_2^2 / \sigma_2^2} \sim F(n_1-1, n_2-1)$。$\frac{\sigma_1^2}{\sigma_2^2}$ 的 $1-\alpha$ 置信区间为：$(\frac{S_1^2}{S_2^2} \frac{1}{F_{\alpha/2}(n_1-1, n_2-1)}, \frac{S_1^2}{S_2^2} F_{\alpha/2}(n_2-1, n_1-1))$。

### 6.6 单侧置信限 ★★★☆☆
- **概念**：有时我们只关心参数的**上限**或**下限**。形如 $P\{\theta > \underline{\theta}\} = 1-\alpha$ 的 $\underline{\theta}$ 称为**置信下限**；形如 $P\{\theta < \bar{\theta}\} = 1-\alpha$ 的 $\bar{\theta}$ 称为**置信上限**。统称**单侧置信限**。
- **方法**：与双侧置信区间类似，使用枢轴量法，但在确定临界值时，只需考虑分布的一侧尾部概率为 $\alpha$。
    - **例如**：求正态总体均值 $\mu$（$\sigma^2$未知）的置信水平为 $1-\alpha$ 的**单侧置信下限**。枢轴量 $T = \frac{\bar{X} - \mu}{S / \sqrt{n}} \sim t(n-1)$。由 $P\{T < t_{\alpha}(n-1)\} = 1-\alpha$，可解得 $\mu > \bar{X} - t_{\alpha}(n-1) \frac{S}{\sqrt{n}}$，故置信下限为 $\underline{\mu} = \bar{X} - t_{\alpha}(n-1) \frac{S}{\sqrt{n}}$。
- **应用**：在可靠性工程、质量控制等领域应用广泛，如评估产品寿命的下限。
