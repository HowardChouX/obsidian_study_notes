# 第九章 消费者态度的形成与改变

## 第一节 消费者态度概述

### 一、态度的定义与特征
**态度**是指消费者对某一事物或观念所持有的正面或负面的认识上的评价，情感上的感受和行为上的倾向[1]。态度具有以下特征[1]：
1. **社会化**：态度是在社会环境中形成和发展的
2. **适应性**：态度具有适应功能
3. **自我防御**：态度具有自我防御功能
4. **知识或认识功能**：态度具有知识或认识功能
5. **价值表达功能**：态度具有价值表达功能

### 二、态度在消费者行为学框架中的位置
在消费者行为学的基本框架中，消费者态度属于**个体与心理因素**的重要组成部分[1]。具体来说，消费者态度与以下因素密切相关[1]：
- 消费者资源
- 需要与动机
- 消费者知觉
- 学习与记忆
- 个性、自我概念与生活方式

### 三、态度的作用
态度对消费者行为具有重要影响[1]：
1. **影响消费者对产品、品牌的判断和评价**[1]
2. **影响消费者的学习兴趣与学习效果**[1]
3. **影响消费者购买意向**[1]

### 四、课堂讨论与头脑风暴
**头脑风暴**[1]：给出你们对Dog的态度，"狗咬人"事件频发后，你们的态度发生变化吗？解释下什么叫"态度"？

**课堂讨论**[1]：消费者对某一品牌强烈的好感的态度，你认为是如何形成的？

## 第二节 消费者态度的测量

### 一、瑟斯顿等距量表
瑟斯顿等距量表是一种经典的态度测量方法，具体步骤如下[7]：
1. **搜集意见**：搜集人们对某一态度对象的各种意见
2. **归类评价**：邀请若干评判人员将上述陈述意见归类为若干组
3. **筛选陈述**：对各陈述意见进行筛选，形成20条左右意义明确的陈述
4. **被试判断**：要求被试对20条陈述意见或其中的一部分进行判断，赞成打勾，不赞成打叉

### 二、李克特量表
李克特量表采用肯定或否定两种陈述，要求被试对各项陈述意见在1-5或1-7的等级量表上自我报告他对陈述意见的赞同程度[7]。例如：
- **示例陈述**：长虹电视清晰度很高
- **评分等级**：极同意、同意、中立、反对、极反对

### 三、语义差别量表
语义差别量表用于分析某一主题概念的语意，确定一些相应的关联词，每个维度都有几对反义形容词[7]。包括三个维度：
1. **情感或评价维度**[7]
2. **力度维度**[7]
3. **活动维度**[7]

**示例**：花店A vs 花店B的评价[7]
| 维度 | 花店A | 花店B |
|------|-------|-------|
| 价格 | 昂贵 vs 便宜 | |
| 选择 | 选择多 vs 选择少 | |
| 可靠性 | 可靠 vs 不可靠 | |
| 友好度 | 友好 vs 不友好 | |
| 时尚度 | 时髦 vs 保守 | |
| 便利性 | 方便 vs 不方便 | |

### 四、行为反应测量法
行为反应测量法包括多种具体方法[7]：
1. **距离测量法**[7]
2. **生理反应测量法**[7]
3. **任务完成法**[7]

## 第三节 消费者态度形成的理论

### 一、学习论
学习论认为人的态度是后天习得的，主要通过以下方式形成[4]：
1. **联想**[4]
2. **强化**[4]
3. **模仿**[4]

具体过程包括[4]：
- **顺从**：表面上的接受
- **认同**：情感上的接受
- **内化**：真正从内心深处相信并接受他人的观点
- **信息的学习**[4]
- **情感的转移**[4]

### 二、诱因论
诱因论认为态度的形成是在权衡利弊之后做出的抉择[4]。具体表现为：
1. **趋近**：主动、积极对诱因冲突进行周密计算后做出选择[4]
2. **回避**[4]

**主观效用公式**：$U = V \times P$[4]
- $U$：主观效用
- $V$：预期后果的价值
- $P$：预期后果出现的概率

### 三、认知相符论
认知相符论包括平衡理论和认知-情感相符理论[4]。

#### 1. 平衡理论
平衡理论通过P-X-O三角模型分析态度关系[4]：
- **P**：个体
- **X**：态度对象
- **O**：相关影响人员

**平衡状态与不平衡状态**[4]：
- **平衡状态**：P-X-O三者关系协调一致
- **不平衡状态**：P-X-O三者关系存在矛盾

**恢复平衡的方法**[4]：
1. 改变相关影响人员的态度
2. 对相关影响人员进行劝说
3. 将三角系统中某两个因素转变为无关联
4. 对某两个因素的关系做出新的归因或解释

#### 2. 认知-情感相符理论
认知-情感相符理论强调态度由认知和情感两个成分构成[4]。

#### 3. 认知失调论
认知失调论认为当两个认知之间出现逻辑上的不一致时，会产生认知失调[16]。例如：明知沉溺于网络游戏会影响学业，但仍然痴迷不能自拔[5]。

**恢复平衡的方法**[16]：
1. 改变其中一个认知
2. 改变行为
3. 在不改变原来两个认知因素的条件下增加新的认知

### 四、自我知觉理论
自我知觉理论认为个体通过观察自己的行为来推断自己的态度[16]。例如：
- **提问**："你爱弟弟吗？"[16]
- **回答**："我总和弟弟一起玩，所以我是爱弟弟的"[16]

### 五、认知反应理论
认知反应理论认为消费者对营销对象态度的改变，取决于认知反应的类型与强度[16]。包括：
1. **反驳论点**[16]
2. **支持论点**[16]
3. **来源贬损**[16]

### 六、加工可能性理论
加工可能性理论提出了态度改变的两条路径[16]：
1. **中心路径**：认知处理的性质（最初态度、论点质量）[16]
2. **边缘路径**：有边缘线索出现时（正面或负面情感、具有吸引力的模型、专家、支持论点数目）[16]

**加工可能性模型的关键问题**[16]：
1. 处理信息的动机高吗？（个人相关性、认知需求、个人责任等）
2. 处理信息的能力强吗？（分心、重复、先前知识、信息可理解性等）

## 第四节 消费者态度的改变

### 一、态度改变的说服模型
态度改变的说服模型包括以下要素[6]：
1. **外部刺激**：传递者、传播特征、情境因素[6]
2. **目标靶**：信奉程度、介入程度、人格因素[6]
3. **中介过程**：学习、感情迁移、相符机制、反驳[6]
4. **结果**：态度改变、信源贬损、信息曲解、掩盖拒绝[6]

### 二、传递者对消费者态度改变的影响
传递者的可信度和喜爱程度对态度改变有重要影响[6]：

#### 1. 可信度[6]
- 找有专长、有地位和受人尊敬、值得信赖的人物传递信息
- 提升公司的影响和声誉，变为值得信任的信息传递者
- 同一信息能以多种独立来源获得认证

#### 2. 喜爱程度[6]
- 外表的吸引力
- 相似性
- 参照群体

### 三、传播特征与消费者态度改变
传播特征对态度改变的影响包括[6]：
1. **与原有态度的差异**[6]
2. **论点的质量或强度**[6]
3. **恐惧的唤起**[6]
4. **幽默的运用**[6]
5. **单面论述与双面论述**[6]

### 四、目标靶的特征
目标靶的特征影响态度改变的效果[6]：

#### 1. 信奉程度[6]
- 承诺会随是否采取了行动而强化
- 承诺会随他在公开场合表明其立场与态度而强化

#### 2. 介入程度[6]
- 介入程度较低，更容易被说服
- 介入程度：个人相关性、反应性介入

#### 3. 人格因素[6]
- 自尊、智力、性别差异等
- 低自尊的人更容易被说服

#### 4. 性别差异[6]
- 差异在于各自擅长的领域

### 五、情境因素与消费者态度改变
情境因素对态度改变的影响包括[6]：
1. **预先警告**[6]
2. **分心**[6]
3. **预防注射**[6]
4. **重复**[6]

### 六、态度如何随时间而变化
态度随时间变化的特点[6]：
1. **图式**[6]
2. **态度客体极端性**[6]
3. **睡眠效应**[6]
4. **"警告"持久性**[6]

## 第五节 消费者态度及行为的预测

### 一、客体态度模型
客体态度模型用于预测消费者对客体的整体态度，公式为[6][33]：
$$A_o = \sum_{i=1}^{n} b_i e_i$$

其中：
- $A_o$：主体对客体O的整体态度
- $I$：客体所具有的第i个重要属性
- $n$：客体具有的重要属性的个数
- $b_i$：消费者对客体具有属性i的信念强度
- $e_i$：对属性i的好恶程度

### 二、行为意向模型
行为意向模型用于预测消费者的行为意向，公式为[33]：
$$B \approx BI = W_1(AB) + W_2(SN)$$

其中：
- $B$：行为
- $BI$：行为意向
- $AB$：对行为B的态度
- $SN$：主观规范
- $W_1$：态度对行为意向的影响权重
- $W_2$：主观规范对形成行为意向的影响权重

**态度计算公式**[33]：
$$AB = \sum_{i=1}^{n} b_i e_i$$

**主观规范计算公式**[33]：
$$SN = \sum_{j=1}^{n} (NB_j)(MC_j)$$

### 三、计划行为理论预测行为的三要素
根据课程作业要求，计划行为理论预测行为的三要素包括[1]：
1. **行为态度**：个体对执行某项行为的态度
2. **主观规范**：个体感知到的社会压力
3. **知觉行为控制**：个体感知到的执行行为的难易程度

### 四、消费者行为与态度不一致的影响因素
消费者行为与态度不一致的影响因素包括[33]：
1. **购买动机**[33]
2. **购买能力**[33]
3. **态度强度**[33]
4. **情境因素**[33]
5. **测度问题**[33]
6. **态度测量与行动之间的延滞**[33]
7. **个人因素**[33]
