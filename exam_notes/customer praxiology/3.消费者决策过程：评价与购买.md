# 第三章 消费者决策过程：评价与购买

## 第一节 购买前的评价

### 一、评价标准的概念与重要性
**评价标准**是消费者在选择备选品时所考虑的产品属性或特征。这些特征与消费者在购买中追求的利益、付出的代价直接相关
**课堂讨论**：购买啤酒（或饮料）时，你对各品牌的评价标准都有哪些?
### 二、购买前的评价过程框架
购买前的评价过程包含三个关键要素：
1. **评价标准**：消费者考虑的产品属性或特征
2. **标准相对重要性**：各评价标准的权重分配
3. **备选产品在每一评价标准上的表现**：各品牌在各标准上的得分

### 三、啤酒评价标准的具体分析
以啤酒为例，消费者可能考虑的评价标准包括[4]：
- 口感
- 价格
- 酿造工艺
- 酒精含量
- 产地
- 包装
**啤酒评价标准及重要性权数示例**：

| 评价标准   | 重要性权数   |
| ------ | ------- |
| 价格     | 20      |
| 口感     | 30      |
| 包装     | 5       |
| 制作工艺   | 15      |
| 产地     | 15      |
| 酒精含量   | 15      |
| **总计** | **100** |

### 四、评价标准的重要性权重
标准相对重要性反映了消费者对不同评价标准的重视程度。例如：
- 全国品牌可能更受年轻/有活力消费者青睐
- 地方品牌可能更受年长/保守消费者欢迎

### 五、备选产品在各标准上的表现
需要确定备选产品在每一评价标准上的具体表现。例如：
```
燕京啤酒的酿造工艺精良 [1 2 3 4 5]
燕京啤酒的口感良好 [1 2 3 4 5]
燕京啤酒的包装精美 [1 2 3 4 5]
燕京啤酒的价格合理 [1 2 3 4 5]
```

**随堂测试**：为什么消费者在评价或判断产品品质时，经常使用替代指标？

### 六、选择规则（决策规则）
消费者使用不同的选择规则来做出最终决策

#### 1. 连接式规则
为每个评价标准设定最低接受标准，只有所有标准都达到最低要求的品牌才会被考虑.

**示例**：啤酒选择的最低标准

| 评价标准 | 接受最低标准 |
| ---- | ------ |
| 口感   | 4      |
| 价格   | 4      |
| 包装   | 1      |
| 产地   | 3      |
| 酿造工艺 | 3      |
| 酒精含量 | 3      |

#### 2. 重点选择规则
为关键标准设定更高的要求，其他标准可以适当放宽

#### 3. 补偿式选择规则
各标准的得分可以相互补偿，一个标准得分低可以通过其他标准得分高来弥补

**示例**：补偿式选择规则的权重分配[48]


| 评价标准   | 权重      | 得分  |
| ------ | ------- | --- |
| 口感     | 30      |     |
| 价格     | 25      |     |
| 酿造工艺   | 15      |     |
| 酒精含量   | 10      |     |
| 产地     | 15      |     |
| 包装     | 5       |     |
| **总分** | **100** |     |

### 七、复杂决策的评价方法
**方法一：做一把"尺子"**
1. **确定关键要素**：例如饮料选择的关键要素包括品牌、口味、包装、价格
2. **量化打分**：为各备选产品在各关键要素上打分
3. **加权汇总**：根据各要素的重要性进行加权计算

**示例**：一个饮品公司研发了几款新品，想知道哪一款会更畅销。直接找100个消费者去品尝，哪款更受欢迎就投产哪个

## 第二节 购买过程

### 一、从评价到购买决策
当消费者有意向购买某品牌产品后，是否立即采取行动购买呢？这取决于多种因素

### 二、购物影响因素——购买情境
购买过程受到多种情境因素的影响

#### 1. 购买任务
- 例如：购买礼品 vs. 购买自用产品
#### 2. 社会环境
- 他人是否在场、彼此互动
#### 3. 时间因素
- 时间充裕程度、活动发生时机
#### 4. 先前状态
- 心情、暂时型情绪（如焦虑、兴奋）、状态（疲劳、备有现金）

### 三、购物影响因素——购物环境
**购物环境**包括：
- **狭义**：购物时的物质环境
- **广义**：同时包括销售人员的着装、仪容和与消费者的互动
**物质环境**是指构成消费者情境的有形物质因素，如地理位置、气味、音响、灯光、温度、商品周围的物质等

### 四、如何影响消费者的实际购买？
**情境因素**，如商店接触、产品（品牌）接触、时间压力，是如何影响消费者决策的？

**影响消费者实际购买的因素包括**：
1. **他人态度**
   - 例如：我舅舅（亲密关系）在啤酒公司工作（权威性），他坚决反对（否定的激烈程度）我购买青岛啤酒
1. **风险感知**
   - 例如：因为他说，之前有消费者买了青岛啤酒，结果发现啤酒里面有头发（风险，纯属虚构）
1. **意外情况或意外事件**
   - 例如：但是我单位今天正好发了超市购物卡（与消费者有关）
   - 例如：去了超市，发现青岛啤酒买二送一（与市场营销活动有关）

4. **追求多样性**
5. **购买冲动性**
6. **购买支付方式**

## 第三节 店铺选择与店内行为

### 一、店铺选择的影响因素
**到哪里去购买啤酒呢？**影响店铺选择的因素包括：

#### 1. 商店的位置与规模
#### 2. 商店的形象
#### 3. 零售店广告

### 二、商店形象的构成层面
商店形象的构成包括多个层面

| 构成层面 | 每一层面的构成内容            |
| ---- | -------------------- |
| 商品   | 品质、选择范围、式样、价格        |
| 服务   | 分期付款计划、销售人员、退货、信用、送货 |
| 主顾   | 顾客类型                 |
| 硬件设施 | 洁净、商店布局、购物便利、吸引力     |
| 方便性  | 店铺位置、停车条件            |
| 促销   | 广告                   |
| 店堂气氛 | 温馨、有趣、兴奋、舒适          |
| 机构声誉 | 交易后感受满意              |

### 三、影响品牌选择和产品购买数量的店内因素
店内因素对消费者的最终购买决策有重要影响
- 零售店广告
- 顾客体验地图

### 四、顾客体验地图：宜家家居与峰终定律
**顾客体验地图**是分析消费者在店铺内体验的重要工具。例如旺哥说营销中分析的宜家家居与峰终定律。

### 五、店铺选择与消费者特征——购物导向
消费者根据其购物导向可以分为不同类型：
1. **不活跃型购物者**
2. **活跃型购物者**
3. **服务型购物者**
4. **传统型购物者**
5. **易变型购物者**
6. **价格型购物者**
7. **过渡型购物者**

### 六、电商VS实体零售业
近年来，电商对实体零售业冲击巨大。在2012年CCTV经济年度人物颁奖盛典上，马云对王健林说：告诉你一个好消息，电商不可能完全取代零售行业，同时也告诉你一个坏消息，它会基本取代你们！

**马云与王健林的1亿赌局**
- 如果2022年中国的电子商务占零售的份额不到50%，马云将输给王健林1个亿
- 反之王健林将输给马云1个亿

**赌局结果**：
10年赌期还未过半，马云和王健林已握手言和。马云在2016年的云栖大会上宣称以后阿里巴巴将不再提"电商"这个词，要打造"新零售"；而万达却悄悄上线了自己的电商平台——飞凡网。

**新零售概念**
马云在演讲中称，纯电商时代过去了，未来十年是新零售的时代，未来线上线下必须结合起来。电商与线下实体商业，应该由原先的独立、冲突，走向混合、融合，通过精准化、体验为主的模式，去了解消费者，满足并引导消费需求，达到消费升级。
