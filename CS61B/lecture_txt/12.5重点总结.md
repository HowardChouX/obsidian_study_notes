## 异常处理

- **抛出异常**：使用 `throw new RuntimeException("message")` 语法
- **作用**：处理意外行为，确保代码合理运行


## 迭代机制

### Iterator vs Iterable

|**特性**|Iterator|Iterable|
|---|---|---|
|**接口方法**|`hasNext()`, `next()`|`iterator()`|
|**功能**|遍历元素|提供迭代器|
|**关系**|由Iterable对象生成|容器类的基础接口（如ArrayList）|

```java
// Iterable 示例
public class MyList implements Iterable<String> {
    public Iterator<String> iterator() {
        return new MyIterator();
    }
}
```

## 对象方法

### toString()

- **作用**：返回对象字符串表示形式
- **使用场景**：调试时自动调用（如 `System.out.println(obj)`）

### equals() vs ==

|**比较方式**|内存地址|内容相等|
|---|---|---|
|`==`|✅|❌|
|`equals()`|❌|✅|

```java
// 正确覆盖 equals 方法
public class Stone {
    int weight;
    
    @Override
    public boolean equals(Object o) {
        return this.weight == ((Stone) o).weight;
    }
}
```

> **重要提示**：覆盖equals()时必须同时覆盖hashCode()方法（后续章节详解）
